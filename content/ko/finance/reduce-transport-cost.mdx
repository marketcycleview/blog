교통비, 생각보다 많이 나가고 있어요.

대중교통만 써도 월 5~10만원, 자차 있으면 **월 30~80만원**까지.

"차 있으니까 편하잖아" → 맞아요. 근데 **그 편함에 월 50만원 내고 있다**는 걸 알고 계신가요?

## 교통비 현황 파악

### 대중교통 vs 자차

| 항목 | 대중교통 | 자차 (국산 중형) |
|------|---------|----------------|
| 월 교통비 | 5~10만원 | 40~80만원 |
| 연 교통비 | 60~120만원 | 480~960만원 |
| 초기 비용 | 0원 | 2,000~4,000만원 |

### 자차 유지비 상세 (국산 중형 기준)

| 항목 | 월 비용 | 비고 |
|------|--------|------|
| 할부금/리스료 | 30~50만원 | 없으면 0원 |
| 자동차 보험 | 8~15만원 | 연령/경력별 |
| 유류비 | 12~20만원 | 월 1,000~1,500km |
| 주차비 | 5~15만원 | 지역별 차이 큼 |
| 정비/소모품 | 3~5만원 | 월 환산 |
| 세차 | 1~3만원 | |
| 자동차세 | 2~4만원 | 월 환산 |
| 톨게이트/벌금 | 1~3만원 | |
| **합계** | **62~115만원** | 할부 포함 |
| **합계** | **32~65만원** | 할부 제외 |

## 대중교통 절약법

### 1. 정기권 활용

| 정기권 | 가격 | 효과 |
|--------|------|------|
| 기후동행카드 (서울) | 65,000원 | 지하철+버스 무제한 |
| K-패스 | 사용액의 20~53% 환급 | 전국 |

**K-패스 환급률:**

| 대상 | 환급률 |
|------|--------|
| 일반 | 20% |
| 청년 (19~34) | 30% |
| 저소득층 | 53% |

월 10만원 교통비면:
- 일반: 2만원 환급 = **실질 8만원**
- 청년: 3만원 환급 = **실질 7만원**

### 2. 환승 할인 최대 활용

```
버스 → 지하철: 환승 할인 (30분 이내)
지하철 → 버스: 환승 할인 (30분 이내)
버스 → 버스: 환승 할인 (30분 이내, 같은 노선 제외)
```

### 3. 카드 교통비 할인

| 카드 | 교통비 할인 |
|------|-----------|
| 신한 Mr.Life | 대중교통 10% |
| KB국민 탄탄대로 | 대중교통 10% |
| 우리카드 가나다라 | 대중교통 10% |

## 자차 유지비 절약법

### 1. 자동차 보험료 줄이기

| 방법 | 절약 효과 |
|------|---------|
| 다이렉트 보험 | 10~20% |
| 마일리지 특약 | 연 2~5만원 |
| 블랙박스 할인 | 2~5% |
| 무사고 할인 | 매년 누적 |
| 자녀 할인 | 5~10% |

**다이렉트 보험만 바꿔도 연 10~20만원 절약.**

### 2. 유류비 줄이기

| 방법 | 절약 효과 |
|------|---------|
| 셀프주유소 | 리터당 50~100원 |
| 주유 앱 (오피넷) | 최저가 주유소 검색 |
| 주유 할인 카드 | 리터당 50~150원 |
| 경제속도 (80~100km) | 연비 10~15% 향상 |
| 타이어 공기압 체크 | 연비 3~5% 향상 |

**월 1,200km 주행 기준:**
- 리터당 100원 절약 × 80L = **월 8,000원 절약**

### 3. 주차비 줄이기

| 방법 | 효과 |
|------|------|
| 월 주차 계약 | 일일 대비 30~50% |
| 공영주차장 이용 | 민영 대비 50% |
| 회사 주차장 | 무료~저렴 |
| 아파트 주차장 | 관리비 포함 |

### 4. 정비비 줄이기

| 방법 | 효과 |
|------|------|
| 셀프세차 | 월 2~3만원 절약 |
| 동네 정비소 (대형 X) | 공임 30~50% |
| 소모품 직접 교체 | 에어필터, 와이퍼 |
| 정기 점검 (예방) | 큰 수리비 방지 |

## 자차 vs 대중교통 vs 카셰어링

### 이용 빈도별 추천

| 주간 이동 | 추천 | 월 비용 |
|---------|------|--------|
| 매일 출퇴근 (대중교통 가능) | 대중교통 | 5~10만원 |
| 매일 출퇴근 (대중교통 불가) | 자차 | 40~60만원 |
| 주말만 | 카셰어링 | 8~15만원 |
| 월 2~3회 | 카셰어링/택시 | 5~10만원 |

### 카셰어링 비용

| 서비스 | 시간당 | 12시간 | 24시간 |
|--------|--------|--------|--------|
| 쏘카 | 6,000~10,000원 | 50,000~70,000원 | 70,000~100,000원 |
| 그린카 | 5,000~9,000원 | 45,000~65,000원 | 65,000~90,000원 |

**월 2회 12시간 이용**: 약 10~14만원
**자차 유지**: 약 40~60만원

**월 2회 이하면 카셰어링이 압도적으로 유리.**

## "차 팔아야 하나요?" 판단 기준

### 팔아야 하는 경우

| 기준 | 해당 |
|------|------|
| 주간 주행 3회 미만 | ✅ |
| 대중교통으로 출퇴근 가능 | ✅ |
| 유지비가 소득의 20% 이상 | ✅ |
| 할부금 부담됨 | ✅ |

**4개 중 3개 이상이면 차 없는 게 경제적.**

### 유지해야 하는 경우

| 기준 | 해당 |
|------|------|
| 대중교통 불편한 지역 | ✅ |
| 업무용 필수 | ✅ |
| 자녀 등하원 | ✅ |
| 유지비 부담 없음 | ✅ |

## 자주 묻는 질문

### Q. 기후동행카드 vs K-패스?

| 비교 | 기후동행카드 | K-패스 |
|------|-----------|--------|
| 지역 | 서울 | 전국 |
| 방식 | 정액제 (월 65,000원) | 환급 (20~53%) |
| 유리한 경우 | 월 교통비 8만원+ | 월 교통비 8만원 미만 |

**서울에서 월 8만원 넘게 쓰면 기후동행카드, 그 외는 K-패스.**

### Q. 전기차가 유지비 싸나요?

**유류비는 싸지만 전체 유지비는 비슷해요.**

| 비교 | 내연기관 | 전기차 |
|------|---------|--------|
| 연료비 | 월 15만원 | 월 4만원 |
| 보험료 | 월 10만원 | 월 13만원 |
| 정비비 | 월 3만원 | 월 2만원 |
| 감가상각 | 보통 | 빠름 |

### Q. 자전거/킥보드 출퇴근?

**5km 이내면 좋은 선택이에요.**

| 수단 | 장점 | 단점 |
|------|------|------|
| 자전거 | 무료, 건강 | 날씨 영향, 땀 |
| 전동킥보드 | 빠름, 편함 | 월 3~5만원 |
| 따릉이 | 저렴 (연 3만원) | 서울 한정 |
