"부모님이 넣어준 보험 매달 나가고 있어요."

보험료, 확인 안 하고 넘기는 사람 많아요. 근데 **월 15만원 넘으면 과한 거예요.**

중복 보험, 불필요한 특약, 저축성 보험 정리하면 **월 3~10만원** 절약 가능해요.

## 정리해야 할 보험

### 1. 저축성 보험

**적금보다 수익 낮고, 해지하면 손해 보는 구조.**

| 비교 | 저축성 보험 | 적금 |
|------|-----------|------|
| 수익률 | 연 1~2% | 연 3~4% |
| 중도 해지 | 원금 손실 | 이자만 손실 |
| 사업비 | 10~20% 떼감 | 없음 |

**가입 3년 미만**: 해지 시 원금 손실 크니 유지/감액 검토
**가입 5년 이상**: 환급금 확인 후 해지 고려

### 2. 종신보험 (미혼/무자녀)

종신보험 = **내가 죽으면 유족에게 돈 주는 보험.**

| 상황 | 필요성 |
|------|--------|
| 미혼, 부양가족 없음 | 불필요 |
| 기혼, 자녀 어림 | 필요 (정기보험 추천) |
| 기혼, 자녀 성인 | 불필요 |

**미혼이면 종신보험 필요 없어요.** 월 5~15만원 절약.

### 3. 중복 실비보험

**실비보험 2개 이상? 의미 없어요.**

실비는 실제 치료비만 보장하기 때문에, 2개 가입해도 **1개만 청구 가능.**

→ 가장 좋은 조건 1개만 유지, 나머지 해지

### 4. 과도한 특약

| 불필요한 특약 | 이유 |
|-------------|------|
| 입원일당 | 실비로 대부분 커버 |
| 수술비 (소액) | 실비로 커버 |
| 통원비 | 실비로 커버 |
| 골절진단금 | 필요성 낮음 |

## 20~30대 필수 보험 (이것만!)

| 보험 | 월 보험료 | 이유 |
|------|---------|------|
| 실비보험 | 3~5만원 | 의료비 핵심 |
| 암보험 (선택) | 2~3만원 | 고액 치료비 대비 |
| 운전자보험 (운전 시) | 1~2만원 | 사고 시 형사 비용 |
| **합계** | **6~10만원** | |

**월 10만원이면 충분해요.** 15만원 넘으면 과한 거예요.

## 보험 리모델링 순서

### Step 1: 현황 파악

- [ ] 보험다모아에서 모든 보험 조회
- [ ] 보험별 월 보험료 확인
- [ ] 총 보험료 계산

### Step 2: 불필요 보험 찾기

| 질문 | Yes면 |
|------|-------|
| 저축성 보험? | 해지 검토 |
| 미혼인데 종신보험? | 해지 |
| 실비 2개 이상? | 1개 해지 |
| 특약이 과도? | 특약 정리 |

### Step 3: 해지 순서

```
1순위: 저축성 보험 (가장 비효율)
2순위: 종신보험 (미혼/무자녀)
3순위: 중복 실비
4순위: 불필요 특약
```

### Step 4: 해지 전 확인

| 확인 항목 | 방법 |
|---------|------|
| 해지환급금 | 보험사 앱/전화 |
| 가입 기간 | 3년 미만이면 손실 큼 |
| 건강 상태 | 재가입 어려울 수 있음 |

**중요**: 실비보험은 새로 가입하면 4세대 실비 (자기부담금 높음). 기존 실비가 좋으면 **절대 해지하지 마세요.**

## 자주 묻는 질문

### Q. 부모님이 넣어준 보험 해지해도 되나요?

**보험 내용 확인 후 판단하세요.**

부모님이 넣어준 보험 중:
- 실비보험 → **절대 유지** (옛날 실비가 더 좋음)
- 종신/저축성 → 해지 검토
- 암보험 → 유지 (보험료 저렴)

### Q. 해지하면 손해 아닌가요?

**이미 낸 돈은 돌아오지 않아요.** 그건 매몰비용이에요.

```
종신보험 월 8만원 × 앞으로 30년 = 2,880만원 추가 납입
해지환급금: 500만원 돌려받음

유지 비용: 2,880만원
해지 후 절약: 2,880만원 - 500만원 = 2,380만원 절약
```

### Q. 보험 없으면 불안한데요?

**실비보험 하나면 대부분 커버돼요.**

| 상황 | 실비 커버 |
|------|---------|
| 감기 진료 | ✅ |
| 입원 | ✅ |
| 수술 | ✅ |
| MRI/CT | ✅ |

**실비 하나로 충분. 나머지는 선택이에요.**

### Q. 보험 리모델링 어디서 받나요?

| 방법 | 비용 | 주의점 |
|------|------|--------|
| 보험다모아 (직접) | 무료 | 가장 객관적 |
| 금융감독원 상담 | 무료 | 1332 전화 |
| 보험 설계사 | 무료 | 자기 회사 상품 추천 주의 |
| 독립 FA | 유/무료 | 상대적으로 객관적 |
