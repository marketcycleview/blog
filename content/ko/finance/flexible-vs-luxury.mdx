"커피가 낭비야? 운동이 낭비야?"

이 질문에 정답은 없어요. 근데 **기준**은 있어요.

같은 5만원이라도 **월급 200만원의 5만원**과 **월급 500만원의 5만원**은 의미가 달라요. 금액이 아니라 **비율**로 판단해야 해요.

## 여유비와 사치비의 차이

### 정의

| 구분 | 여유비 | 사치비 |
|------|--------|--------|
| 뜻 | 삶의 질을 위한 소비 | 과하거나 불필요한 소비 |
| 특징 | 즐거움 + 통제 가능 | 후회 + 통제 어려움 |
| 예산 | 소득의 15~25% | 0%가 목표 |
| 기분 | "잘 쓴 돈" | "왜 샀지..." |

### 핵심 기준 3가지

**1. 비율 기준**: 소득의 20~25% 이내인가?

```
월급 300만원 → 여유비 60~75만원
이 안에서 쓰면 여유비, 넘으면 사치비
```

**2. 반복 기준**: 습관적으로 쓰는가?

```
월 1회 좋은 레스토랑 → 여유비
매일 배달 음식 → 낭비
```

**3. 후회 기준**: 쓰고 나서 후회하는가?

```
"오늘 기분 좋았다" → 여유비
"왜 또 샀지..." → 사치비
```

## 항목별 낭비 기준

### 카페/커피

| 패턴 | 월 비용 | 판정 |
|------|--------|------|
| 회사 커피 (무료) | 0원 | - |
| 편의점 커피 | 2~3만원 | 여유비 |
| 주 2~3회 카페 | 4~6만원 | 여유비 |
| 매일 카페 (아아+디저트) | 10~15만원 | 과소비 |
| 매일 스타벅스 라떼 | 15만원+ | 사치비 |

**적정선**: 소득의 2~3% (300만원이면 **월 6~9만원**)

### 외식/배달

| 패턴 | 월 비용 | 판정 |
|------|--------|------|
| 주 1회 외식 | 8~12만원 | 여유비 |
| 주 2회 외식 + 주 1회 배달 | 20~25만원 | 여유비 |
| 주 3회+ 외식 + 주 2회+ 배달 | 40만원+ | 과소비 |
| 거의 매일 배달 | 60만원+ | 사치비 |

**적정선**: 소득의 5~8%

### 쇼핑/의류

| 패턴 | 월 비용 | 판정 |
|------|--------|------|
| 계절 필수품 (월 환산) | 3~5만원 | 기초생활비 |
| 간간이 원하는 옷 | 5~10만원 | 여유비 |
| 매달 온라인 쇼핑 | 15~25만원 | 과소비 |
| 명품/충동구매 | 30만원+ | 사치비 |

**적정선**: 소득의 3~5%

### 취미/운동

| 패턴 | 월 비용 | 판정 |
|------|--------|------|
| 헬스장 | 5~10만원 | 여유비 (건강 투자) |
| 필라테스/PT | 15~30만원 | 여유비~과소비 |
| 골프 | 30~50만원 | 소득에 따라 |
| 취미 (게임/독서 등) | 3~10만원 | 여유비 |

**적정선**: 소득의 3~5%

### 구독 서비스

| 개수 | 월 비용 | 판정 |
|------|--------|------|
| 1~2개 | 1~2만원 | 여유비 |
| 3~4개 | 3~5만원 | 점검 필요 |
| 5개+ | 7만원+ | 정리 필요 |

**적정선**: 월 3만원 이하

## "이건 낭비야?" 자가 진단

### 체크리스트

아래 중 3개 이상이면 지출 점검이 필요해요.

- [ ] 월말에 항상 돈이 부족하다
- [ ] 카드 명세서 보면 놀란다
- [ ] "왜 샀지?" 싶은 물건이 있다
- [ ] 같은 카테고리에서 매달 지출이 늘고 있다
- [ ] 쓰면서 "다음 달부터 줄여야지" 생각한다
- [ ] 배달 앱 주문 내역이 20건 이상이다
- [ ] 안 쓰는 구독 서비스가 있다
- [ ] 옷장에 태그 안 뗀 옷이 있다

### 진단 결과

| 체크 개수 | 상태 | 조치 |
|---------|------|------|
| 0~1개 | 건강 | 유지 |
| 2~3개 | 주의 | 가계부 2주 작성 |
| 4~5개 | 경고 | 여유비 20% 이하로 줄이기 |
| 6개+ | 위험 | 전면 재검토 필요 |

## 낭비를 줄이는 실전 방법

### 1. 72시간 규칙

**충동구매 방지의 핵심.**

```
사고 싶은 게 생겼다
→ 장바구니에 넣는다
→ 72시간 기다린다
→ 아직도 갖고 싶은가?
   → Yes: 산다 (합리적 소비)
   → No: 삭제 (충동이었음)
```

**효과**: 충동구매 50% 이상 줄어요.

### 2. 월초 봉투 예산

```
여유비 60만원
├── 외식/카페: 20만원 (현금 or 체크카드)
├── 쇼핑: 10만원
├── 취미: 10만원
├── 자유: 20만원
```

**정해진 금액만 쓰기.** 다 쓰면 그 달은 끝.

### 3. 1일 1지출 기록

가계부까진 아니어도, **하루 1번 지출 기록**.

```
1/29: 점심 12,000 + 카페 5,500 + 편의점 3,000 = 20,500원
```

**기록만 해도 의식이 달라져요.** 안 쓰게 돼요.

### 4. "시급 환산" 사고법

**이 물건이 내 노동 몇 시간인지 계산.**

```
월급 300만원 ÷ 209시간 = 시급 약 14,350원

- 5만원 옷 = 3.5시간 노동
- 15만원 신발 = 10.5시간 노동
- 30만원 가방 = 21시간 노동 (약 3일)
```

"이 가방을 위해 3일 일할 가치가 있나?"

## 여유비를 잘 쓰는 법

### 낭비와 투자의 차이

| 지출 | 낭비인가? | 이유 |
|------|---------|------|
| 운동 (헬스) | 투자 | 건강 = 의료비 절감 |
| 책/강의 | 투자 | 성장 = 수입 증가 가능 |
| 친구 만남 | 여유비 | 관계 유지 = 정신 건강 |
| 매일 배달 | 낭비 | 습관적 + 건강에 안 좋음 |
| 안 쓰는 구독 | 낭비 | 돈만 나감 |
| 여행 | 여유비 | 일시적 + 경험 |

### 여유비 활용 우선순위

```
1순위: 건강 (운동, 건강식)
2순위: 관계 (모임, 데이트)
3순위: 경험 (여행, 공연)
4순위: 자기계발 (강의, 책)
5순위: 물질적 소비 (쇼핑)
```

**물건보다 경험에 쓰면 만족도가 더 높아요.**

## 절대 줄이면 안 되는 지출

### 아끼면 오히려 손해

| 항목 | 이유 |
|------|------|
| 건강검진 | 질병 조기 발견 = 치료비 절감 |
| 보험 (실비) | 큰 사고 시 수천만원 차이 |
| 안전 관련 | 가스 점검, 소화기 등 |
| 기본 식사 | 영양 부족 → 건강 악화 |
| 수면 환경 | 매트리스, 침구 → 건강 직결 |

**"아끼다가 더 큰 돈 나가는" 항목은 건드리지 마세요.**

## 자주 묻는 질문

### Q. 카페가 낭비인가요?

**빈도에 따라 달라요.**

| 빈도 | 월 비용 | 판정 |
|------|--------|------|
| 주 1~2회 | 2~4만원 | 여유비 |
| 주 3~4회 | 5~8만원 | 소득에 따라 |
| 매일 | 10~15만원 | 줄여볼 만함 |

**커피를 줄이는 것보다 고정비를 줄이는 게 효과 크다**는 점 잊지 마세요.

### Q. 운동은 사치인가요?

**아니요, 건강 투자예요.**

| 운동 | 월 비용 | 판정 |
|------|--------|------|
| 헬스장 | 5~10만원 | 합리적 |
| PT (주 1회) | 20~30만원 | 소득에 따라 |
| PT (주 3회+) | 50만원+ | 과한 편 |

### Q. 여행은 낭비인가요?

**계획된 여행은 여유비, 충동 여행은 사치비.**

```
3개월 전 예약, 예산 범위 내 → 여유비
"갑자기 떠나자!" + 카드 결제 → 사치비
```

### Q. 여유비 0%로 하면 안 되나요?

**안 돼요. 번아웃 와요.**

극단적 절약 → 스트레스 → 폭발적 소비

**15~20%는 유지하세요.** "잘 쓰는 것"도 능력이에요.
