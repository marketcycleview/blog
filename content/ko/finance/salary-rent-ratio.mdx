"연봉 3000인데 월세 60만원이면 괜찮은 건가요?"

주거비는 생활비에서 **가장 큰 비중**을 차지해요. 이걸 잘못 정하면 저축은 꿈도 못 꿔요.

**기준**: 월 실수령액의 **25% 이하**가 적정 월세예요.

| 2,400만원 | 약 185만원 | 46만원 | 56만원 |
| 3,000만원 | 약 225만원 | 56만원 | 68만원 |

| 4,000만원 | 약 285만원 | 71만원 | 86만원 |
| 4,500만원 | 약 315만원 | 79만원 | 95만원 |
| 5,000만원 | 약 345만원 | 86만원 | 104만원 |
| 6,000만원 | 약 400만원 | 100만원 | 120만원 |
| 7,000만원 | 약 465만원 | 116만원 | 140만원 |

### 25% vs 30%

| 기준 | 저축 가능성 | 삶의 질 |
|------|-----------|--------|
| 20% 이하 | 저축 30%+ 가능 | 약간 불편 |
| **25%** | **저축 25~30% 가능** | **균형** |
| 30% | 저축 20% 정도 | 편함 |
| 35%+ | 저축 어려움 | 위험 |

**25%가 황금 비율.** 30% 넘으면 주거비 때문에 저축이 힘들어져요.

## 월세 vs 전세 판단 기준

### 언제 전세가 유리한가?

| 조건 | 전세 | 월세 |
|------|------|------|
| 거주 기간 3년+ | ✅ | |
| 전세 대출 이자 < 월세 | ✅ | |
| 목돈 있음 | ✅ | |
| 거주 기간 1~2년 | | ✅ |
| 이사 가능성 높음 | | ✅ |
| 목돈 없음 | | ✅ |

### 전세 대출 이자 vs 월세

전세 2억, 대출 70% (1.4억), 금리 3.5% 기준:

| 비교 | 월 비용 |
|------|--------|
| 전세 대출 이자 | 약 41만원 |
| 같은 조건 월세 | 약 55~65만원 |

**전세 대출 이자가 월세보다 싸면 전세가 유리.**

## 주거비 줄이는 방법

### 1. 역세권 5분 → 15분

| 도보 거리 | 월세 차이 |
|---------|---------|
| 역 5분 이내 | 기준가 |
| 역 10분 | -5~10만원 |
| 역 15분 | -10~15만원 |

### 2. 신축 → 구축

| 건물 연식 | 월세 차이 |
|---------|---------|
| 신축 (5년 이내) | 기준가 |
| 준신축 (5~10년) | -5~10만원 |
| 구축 (10년+) | -10~20만원 |

### 3. 보증금 올리고 월세 줄이기

보증금 100만원 올리면 월세 약 3,000~5,000원 줄어요.

| 보증금 | 월세 (예시) |
|--------|-----------|
| 500만원 | 60만원 |
| 1,000만원 | 55만원 |
| 2,000만원 | 48만원 |

### 4. 룸메이트

| 비교 | 혼자 | 2인 셰어 |
|------|------|---------|
| 월세 | 55만원 | 35만원 (각자) |
| 관리비 | 10만원 | 5만원 (각자) |
| **합계** | **65만원** | **40만원** |

## 월세 계약 전 체크리스트

- [ ] 월세가 실수령의 25% 이하인가
- [ ] 관리비 포함 총 주거비 확인했는가
- [ ] 보증금 보호 (전입신고, 확정일자)
- [ ] 공과금 별도인지 포함인지 확인
- [ ] 중개수수료 확인

## 자주 묻는 질문

### Q. 연봉 2400인데 서울에서 살 수 있나요?

**가능하지만 선택이 제한돼요.**

적정 월세: 46만원 이하
→ 서울 외곽 (노원, 도봉, 구로) 또는 경기도

### Q. 월세 30% 넘으면 어떻게 해요?

**3가지 선택지:**
1. 더 저렴한 곳으로 이사
2. 룸메이트/셰어하우스
3. 수입 늘리기

### Q. 관리비도 주거비에 포함해요?

**네, 포함해서 계산하세요.**

```
월세 50만원 + 관리비 10만원 = 실질 주거비 60만원
```
