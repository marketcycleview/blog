"아끼는 게 나을까, 더 버는 게 나을까?"

정답은 **둘 다**예요. 근데 순서가 있어요.

**먼저 짠테크로 새는 돈 막고, 그 다음 돈테크로 불리는 거예요.**

월급 300만원에서 50만원 아끼는 게 먼저, 그 50만원을 불리는 게 다음이에요.

## 짠테크 vs 돈테크 비교

| 구분 | 짠테크 | 돈테크 |
|------|--------|--------|
| 방향 | 지출 줄이기 | 수입 늘리기 |
| 핵심 | 절약, 고정비 관리 | 투자, 부업, 이직 |
| 효과 시점 | 즉시 | 시간 필요 |
| 위험도 | 없음 | 있음 (투자 손실) |
| 한계 | 0원 이하로 못 줄임 | 한계 없음 |
| 필요 자본 | 없음 | 있음 (투자) |

### 핵심 차이

```
짠테크: 월 300만원 → 지출 200만원 → 저축 100만원
돈테크: 월 300만원 + 부수입 50만원 → 저축 150만원

둘 다: 월 300만원 + 50만원 → 지출 200만원 → 저축 150만원
```

## 짠테크 (절약형)

### 짠테크 방법 TOP 10

| 순위 | 방법 | 월 효과 | 난이도 |
|------|------|--------|--------|
| 1 | 알뜰폰 변경 | 4만원 | 1회 작업 |
| 2 | 보험 리모델링 | 5만원 | 1회 작업 |
| 3 | 구독 정리 | 3만원 | 1회 작업 |
| 4 | 배달 줄이기 | 10만원 | 습관 변경 |
| 5 | 도시락/자취 | 10만원 | 습관 변경 |
| 6 | 카드 → 체크카드 | 5~10만원 | 1회 작업 |
| 7 | 교통비 최적화 | 3만원 | 1회 작업 |
| 8 | 무지출 데이 | 5만원 | 주 2~3일 |
| 9 | 공과금 절약 | 2만원 | 습관 변경 |
| 10 | 쇼핑 72시간 규칙 | 변동 | 마인드셋 |

**짠테크만으로 월 30~50만원 절약 가능.**

### 짠테크 장점

- 즉시 효과 (이번 달부터)
- 리스크 없음
- 누구나 가능
- 한 번 바꾸면 자동 (고정비)

### 짠테크 한계

- 줄이는 데 한계 있음
- 삶의 질 저하 가능
- 수입 자체를 늘리진 못함

## 돈테크 (수입 증가형)

### 돈테크 방법

| 방법 | 예상 수익 | 위험도 | 진입 장벽 |
|------|---------|--------|---------|
| 이직/연봉 협상 | 월 50~200만원 | 낮음 | 스킬 |
| 부업 (프리랜서) | 월 30~100만원 | 낮음 | 시간+스킬 |
| 연금저축 세액공제 | 연 91만원 환급 | 없음 | 없음 |
| ETF 적립 투자 | 연 5~10% | 중간 | 공부 필요 |
| 주식 투자 | 변동 | 높음 | 공부 필요 |
| 부동산 투자 | 변동 | 높음 | 자본 필요 |
| 앱테크 | 월 1~5만원 | 없음 | 없음 |

### 돈테크 장점

- 수입에 한계가 없음
- 자산 증식 가능
- 복리 효과

### 돈테크 한계

- 시간이 걸림
- 투자는 손실 위험
- 초기 자본 필요 (투자)

## 소득별 추천 전략

### 월급 200만원 이하: 짠테크 80% + 돈테크 20%

```
핵심: 새는 돈 막기가 우선
- 고정비 총점검 (알뜰폰, 보험, 구독)
- 식비 구조 변경 (자취 + 주 1회 외식)
- 남는 돈 → 비상금 먼저

돈테크: 이직/스킬업 (수입 자체를 늘려야)
```

### 월급 300~400만원: 짠테크 50% + 돈테크 50%

```
핵심: 절약 + 투자 동시 진행
- 고정비 정리 (이미 했으면 유지)
- 연금저축 + IRP (세액공제)
- ETF 적립 시작

짠테크: 생활 수준 유지 (올리지 않기)
돈테크: 연금저축 → ETF → 부업
```

### 월급 500만원+: 짠테크 30% + 돈테크 70%

```
핵심: 자산 배분이 핵심
- 절약은 기본만 (고정비 관리)
- 투자 포트폴리오 구성
- 절세 최적화

짠테크: 라이프스타일 인플레이션 방지
돈테크: 연금저축 + IRP + ISA + ETF + 부동산 검토
```

## 단계별 로드맵

### 1단계: 짠테크 (0~3개월)

- [ ] 고정비 점검 (통신비, 보험, 구독)
- [ ] 통장 쪼개기
- [ ] 자동이체 설정
- [ ] 비상금 시작 (목표: 3개월 생활비)

### 2단계: 짠테크 + 돈테크 (3~6개월)

- [ ] 비상금 목표 50% 달성
- [ ] 연금저축 가입 (월 33만원)
- [ ] 가계부로 지출 최적화
- [ ] 스킬업/부업 탐색

### 3단계: 돈테크 중심 (6개월~)

- [ ] 비상금 완성
- [ ] IRP 추가 가입
- [ ] ETF 적립 시작
- [ ] 이직/연봉 협상 검토

## 자주 묻는 질문

### Q. 짠테크만 하면 부자 될 수 있나요?

**부자는 어렵고, 안정은 가능해요.**

짠테크로 월 50만원 절약 × 10년 = 6,000만원 (이자 포함 7,000만원+)

안정적이지만 "부자"는 아니에요. 부자 되려면 돈테크(투자/사업/이직)가 필요해요.

### Q. 투자부터 해도 되나요?

**비상금 먼저요.**

비상금 없이 투자 → 급할 때 손해 보고 빼야 함
비상금 있고 투자 → 여유 있게 장기 투자 가능

### Q. 앱테크 해볼까요?

**시간 대비 수익 낮아요.**

| 앱테크 | 월 수익 | 시간 |
|--------|--------|------|
| 리워드 앱 | 1~3만원 | 매일 30분 |
| 설문 조사 | 1~2만원 | 매일 20분 |
| 영수증 촬영 | 5천원 | 매일 5분 |

**시간 있으면 스킬업이 수익률 더 높아요.**

### Q. 둘 다 할 시간이 없어요

**짠테크는 시간이 안 들어요.** 한 번 세팅하면 끝.

1. 알뜰폰 변경 (30분)
2. 구독 정리 (15분)
3. 자동이체 설정 (15분)

**총 1시간이면 월 10만원+ 절약.**
