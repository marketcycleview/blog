"월급 받으면 바로 사라져요."

매달 이런 기분이면, 적자 상태일 가능성이 높아요.

적자 = 수입보다 지출이 많은 상태. **카드값이 월급보다 많거나**, 매달 마이너스 통장을 쓰고 있다면 해당돼요.

**30일이면 바꿀 수 있어요.** 아끼라는 얘기가 아니에요. **구조를 바꾸는 거예요.**

## 적자인지 확인하기

### 적자 자가 진단

아래 중 3개 이상이면 적자 상태예요.

- [ ] 월말에 잔고가 0원 근처다
- [ ] 카드값이 월급의 80% 이상이다
- [ ] 마이너스 통장을 쓰고 있다
- [ ] 적금을 깬 적 있다
- [ ] "이번 달만" 하고 매달 반복한다
- [ ] 정확한 월 지출을 모른다
- [ ] 비상금이 100만원 미만이다

### 적자 단계

| 단계 | 상태 | 월 적자 |
|------|------|--------|
| 경미 | 가끔 빠듯 | 0~10만원 |
| 중간 | 매달 부족 | 10~30만원 |
| 심각 | 빚으로 메꿈 | 30만원+ |
| 위험 | 빚이 늘어남 | 카드론/대출 |

## 왜 적자가 되었나?

### 흔한 원인 TOP 5

| 순위 | 원인 | 비율 |
|------|------|------|
| 1 | 주거비 과다 | 35% |
| 2 | 식비/외식 과다 | 25% |
| 3 | 카드 습관성 소비 | 20% |
| 4 | 보험료/구독 과다 | 10% |
| 5 | 빚 이자 부담 | 10% |

### 원인별 진단

**주거비**: 월급의 30% 넘는가?
```
월급 250만원 × 30% = 75만원
월세 80만원이면? → 주거비 과다
```

**식비**: 1인 월 40만원 넘는가?
```
매일 배달: 1.5만원 × 30일 = 45만원
자취 + 주 2회 외식: 30만원
→ 15만원 차이
```

**카드**: 월 결제액이 실수령의 70% 넘는가?
```
실수령 250만원의 70% = 175만원
카드값 200만원이면? → 과소비
```

## 30일 적자 탈출 플랜

### Week 1 (1~7일): 현실 파악

**Day 1~3: 전체 지출 파악**

모든 카드, 통장, 현금 지출을 뽑아보세요.

| 확인 항목 | 방법 |
|---------|------|
| 카드 결제액 | 카드사 앱 → 월별 명세 |
| 이체 내역 | 은행 앱 → 3개월 내역 |
| 현금 지출 | 기억나는 것 메모 |
| 자동이체 | 정기결제 내역 조회 |

**Day 4~5: 카테고리 분류**

| 카테고리 | 3개월 평균 |
|---------|-----------|
| 주거비 | ___만원 |
| 식비 | ___만원 |
| 교통비 | ___만원 |
| 통신비 | ___만원 |
| 보험료 | ___만원 |
| 외식/카페 | ___만원 |
| 쇼핑 | ___만원 |
| 구독 | ___만원 |
| 기타 | ___만원 |
| **합계** | **___만원** |

**Day 6~7: 적자 금액 확인**

```
월 수입: ___만원
월 지출: ___만원
차이: ___만원 (+ 흑자 / - 적자)
```

### Week 2 (8~14일): 고정비 줄이기

**가장 효과 큰 순서대로:**

**1. 통신비 (즉시 효과)**

| Before | After | 절약 |
|--------|-------|------|
| 대형 통신사 7만원 | 알뜰폰 2.5만원 | 4.5만원 |
| TV+인터넷 결합 4만원 | 인터넷만 2만원 | 2만원 |
| **합계** | | **6.5만원** |

**2. 구독 정리 (즉시 효과)**

| Before | After | 절약 |
|--------|-------|------|
| 넷플릭스 17,000 | 유지 | 0 |
| 유튜브 프리미엄 14,900 | 해지 | 14,900 |
| 멜론 10,900 | 해지 | 10,900 |
| 앱 구독 2개 10,000 | 해지 | 10,000 |
| **합계** | | **35,800원** |

**3. 보험 점검 (2주 소요)**

| Before | After | 절약 |
|--------|-------|------|
| 실비 + 암 + 종신 20만원 | 실비 + 암 12만원 | 8만원 |

**Week 2 합계: 약 18만원 절약 가능**

### Week 3 (15~21일): 변동비 구조 바꾸기

**1. 식비 구조 변경**

| Before | After | 절약 |
|--------|-------|------|
| 배달 주 3회 (4.5만원) | 주 1회 (1.5만원) | 12만원 |
| 외식 주 4회 (6만원) | 주 2회 (3만원) | 12만원 |
| **합계** | | **24만원** |

**2. 교통비**

| Before | After | 절약 |
|--------|-------|------|
| 택시 주 2회 | 대중교통 | 6만원 |

**3. 카드 → 체크카드**

신용카드 지갑에서 빼세요. **체크카드만 사용.**

이유:
- 잔고 보이니까 심리적 제한
- 할부 유혹 차단
- 과소비 자동 방지

**Week 3 합계: 약 30만원 절약 가능**

### Week 4 (22~30일): 시스템 구축

**1. 통장 분리**

```
월급 통장
├── 기초생활비 통장 (자동이체)
├── 생활비 통장 (체크카드 연결)
└── 저축 통장 (자동이체)
```

**2. 자동이체 설정**

월급일에 자동으로:
- 월세/관리비 이체
- 적금 이체 (소액이라도!)
- 보험료 이체

**3. 주간 예산 설정**

변동 지출을 주 단위로 관리:

```
월 여유비 40만원
÷ 4주 = 주 10만원

일요일마다 잔액 체크
```

## 적자 금액별 처방전

### 월 10만원 적자

| 처방 | 효과 |
|------|------|
| 알뜰폰 변경 | +4만원 |
| 구독 2개 해지 | +2만원 |
| 배달 주 1회 줄이기 | +4만원 |
| **합계** | **+10만원 → 흑자 전환** |

### 월 30만원 적자

| 처방 | 효과 |
|------|------|
| 알뜰폰 변경 | +4만원 |
| 구독 정리 | +3만원 |
| 보험 정리 | +5만원 |
| 배달/외식 줄이기 | +10만원 |
| 택시 → 대중교통 | +5만원 |
| 쇼핑 월 예산 제한 | +5만원 |
| **합계** | **+32만원 → 흑자 전환** |

### 월 50만원+ 적자

고정비만으로는 부족. **주거비 재검토 필요.**

| 처방 | 효과 |
|------|------|
| 위 모든 항목 | +32만원 |
| 월세 낮은 곳 이사 | +10~20만원 |
| **합계** | **+42~52만원** |

주거비를 못 바꾸면? → **수입 늘리기** (야근수당, 부업, 이직)

## 빚이 있다면: 추가 전략

### 빚 상환 우선순위

| 순위 | 빚 종류 | 금리 | 조치 |
|------|--------|------|------|
| 1 | 카드론/현금서비스 | 15~20% | 최우선 상환 |
| 2 | 리볼빙 | 15~18% | 즉시 해지 |
| 3 | 소비자 대출 | 5~10% | 갈아타기 검토 |
| 4 | 학자금 | 2~3% | 정상 상환 |
| 5 | 전세 대출 | 3~4% | 유지 |

### 리볼빙 해지하세요

**지금 당장요.**

| 리볼빙 | 현실 |
|--------|------|
| "최소 결제만 하면 돼" | 이자가 눈덩이 |
| 50만원 리볼빙 | 연 이자 7~9만원 |
| 200만원 리볼빙 | 연 이자 30~36만원 |

리볼빙은 **빚이 줄지 않는 구조**예요. 해지하고 전액 결제로 바꾸세요.

## 적자 탈출 후: 흑자 유지 비결

### 월 1회 점검

매달 마지막 날:
- [ ] 이번 달 수입/지출 확인
- [ ] 예산 초과 항목 체크
- [ ] 다음 달 조정 사항 메모

### 비상금 만들기

흑자로 바뀌면 첫 목표는 **비상금**.

```
목표: 월 생활비 × 3개월
생활비 150만원 → 비상금 450만원
매달 10만원씩 → 3.75년
매달 30만원씩 → 1.25년
```

### 저축 자동화

```
흑자 10만원 → 적금 자동이체
흑자 20만원 → 적금 10 + 비상금 10
흑자 30만원+ → 적금 15 + 비상금 10 + 투자 5
```

## 자주 묻는 질문

### Q. 가계부 꼭 써야 하나요?

**최소 1개월은 쓰세요.**

앱 추천:
- 뱅크샐러드 (자동 분류)
- 토스 (지출 분석)
- 엑셀 (직접 관리)

1개월만 써도 **"아, 여기서 새고 있었구나"** 알 수 있어요.

### Q. 수입이 적은 게 문제 아닌가요?

**수입이 적어도 적자 아닌 사람 있어요.** 수입이 많아도 적자인 사람도 있고요.

| 월급 200만원 | 지출 180만원 | 흑자 20만원 |
| 월급 400만원 | 지출 420만원 | 적자 20만원 |

**구조가 문제예요.**

### Q. 의지가 약해서 안 돼요

**의지로 하지 마세요. 시스템으로 하세요.**

```
의지: "이번 달은 아껴야지" → 실패
시스템: 체크카드 + 자동이체 + 주간 예산 → 자동으로 관리
```

### Q. 가족이 이해를 못 해요

**숫자로 보여주세요.**

```
"아끼자" → 공감 X
"우리 지금 월 30만원 적자야" → 위기감 ○
"이대로 1년이면 360만원 빚이야" → 행동 ○
```
