"이번 달 결혼식 3개, 축의금만 30만원..."

경조사비는 **예측 불가능한 지출**이에요. 갑자기 몰리면 한 달 예산이 무너지기도 하죠.

2026년 기준 직장인 평균 경조사비 **월 10~15만원**. 근데 이걸 줄이라고 하기엔 인간관계가 걸려 있어서 쉽지 않아요.

핵심은 **"줄이기"보다 "대비하기"**예요.

## 경조사비 줄이는 5가지 방법

### 1. 연간 예산 세우기

**가장 효과적인 방법.**

```
연간 경조사 예산: 150만원
월 적립: 12.5만원 (경조사비 통장에)
```

별도 통장에 매달 자동이체 해두면 갑자기 몰려도 부담 없어요.

### 2. 기준 금액 정하기

**모든 사람에게 같은 금액.**

```
내 기준:
- 직장 동료/아는 사이: 5만원
- 친한 친구: 10만원
- 가족: 집안 기준 따름
```

**일관된 기준이 있으면 고민이 줄어요.**

### 3. 불참 시 축의금 안 보내기

| 상황 | 조치 |
|------|------|
| 참석 | 축의금 |
| 불참 (친하지 않음) | 안 보내도 됨 |
| 불참 (친한 사이) | 축의금 또는 선물 |

**의무감에 모든 청첩장에 돈 보내지 마세요.**

### 4. 공동 선물/공동 축의금

```
직장 동료 결혼 → 팀 단위 공동 축의금
예: 5명이 5만원씩 = 25만원 (1인당 5만원)
```

### 5. 선물 대신 시간

**모든 경조사에 돈이 답이 아니에요.**

| 상황 | 대안 |
|------|------|
| 친구 생일 | 직접 만든 것, 같이 시간 보내기 |
| 집들이 | 실용적인 소품 (3만원 이하) |
| 돌잔치 | 아기 용품 (실용적인 것) |

## 소득별 적정 경조사비

| 월 실수령 | 적정 (월) | 연간 | 비고 |
|---------|---------|------|------|
| 170만원 | 5~8만원 | 60~96만원 | 최소한으로 |
| 250만원 | 8~12만원 | 96~144만원 | |
| 340만원 | 10~15만원 | 120~180만원 | |
| 410만원 | 12~18만원 | 144~216만원 | |

**소득의 3~5%가 적정.** 넘으면 다른 지출을 줄여야 해요.

## 경조사비 거절해도 되는 경우

**"꼭 내야 하나?"** 싶을 때:

| 상황 | 거절 가능 |
|------|---------|
| 1년 이상 연락 안 한 사이 | ✅ |
| 단체 카톡으로 받은 청첩장 | ✅ |
| 불참하는 행사 (안 친한) | ✅ |
| 회사 전체 공지 (개인적 안 친함) | ✅ |

**돈 안 내고 축하 메시지만 보내도 됩니다.** 진짜 관계는 돈으로 유지되지 않아요.

## 자주 묻는 질문

### Q. 축의금 5만원이면 적나요?

**아니요, 2026년 기준 5만원이 표준이에요.** 친한 사이가 아니면 충분해요.

### Q. 안 친한 사람 결혼식 꼭 가야 하나요?

**안 가도 됩니다.** 축하 메시지로 대체 가능해요. 관계 유지가 목적이 아니면 부담 가지지 마세요.

### Q. 경조사비 때문에 적자인데요?

**경조사비 통장 만드세요.** 월 10만원씩 적립해두면 몰리는 달에도 부담 없어요.
