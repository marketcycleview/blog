집주인이 전세를 월세로 바꿀 때 마음대로 높은 전환율을 적용할 수 없습니다.

법으로 **상한선**이 정해져 있어요. 2026년 기준 약 **6.0~6.5%**입니다.

---

## 법정 전환율 상한 공식

```
법정 전환율 상한 = min(10%, 한국은행 기준금리 + 3.5%)
```

10%와 (기준금리 + 3.5%) 중 **작은 값**이 상한입니다.

### 기준금리별 법정 상한

| 한국은행 기준금리 | 기준금리 + 3.5% | 법정 상한 |
|-----------------|---------------|----------|
| 2.0% | 5.5% | **5.5%** |
| 2.25% | 5.75% | **5.75%** |
| 2.5% | 6.0% | **6.0%** |
| 2.75% | 6.25% | **6.25%** |
| **3.0%** | **6.5%** | **6.5%** |
| 3.25% | 6.75% | **6.75%** |
| 3.5% | 7.0% | **7.0%** |

> 기준금리가 6.5% 이상이면 상한이 10%로 적용되지만, 현실적으로 그런 경우는 거의 없습니다.

## 법정 상한의 실제 적용

### 언제 적용되나?

| 상황 | 적용 여부 |
|------|----------|
| 전세 → 월세 전환 시 | **적용** |
| 보증금 감액 + 월세 증액 시 | **적용** |
| 신규 계약 (처음 계약) | **적용** |
| 계약 갱신 시 전환 | **적용** |
| 상가 임대차 | 적용 안 됨 (주택만) |

### 적용 사례

**사례: 전세 3억 → 보증금 1억 + 월세**

| 전환율 | 월세 | 법정 상한(6.25%) 판단 |
|--------|------|---------------------|
| 4.5% | 약 75만 | 적법 |
| 5.5% | 약 92만 | 적법 |
| **6.25%** | **약 104만** | **상한** |
| 7.0% | 약 117만 | **위반** |
| 8.0% | 약 133만 | **위반** |

법정 상한 6.25% 기준, 전세 3억에서 보증금 1억으로 전환 시 월세 상한은 **약 104만원**입니다.

## 상한 초과 시 대처법

### 1. 계약 전 (계약서 작성 전)

| 행동 | 방법 |
|------|------|
| 거부 | 법정 상한을 설명하고 낮춰달라고 요구 |
| 계산 제시 | 현재 기준금리 + 3.5% = 상한을 보여주기 |
| 다른 물건 찾기 | 시세에 맞는 물건으로 변경 |

### 2. 계약 후 (이미 높은 전환율로 계약)

| 단계 | 방법 |
|------|------|
| 1단계 | 집주인에게 법정 상한 초과를 알리고 조정 요청 |
| 2단계 | **주택임대차분쟁조정위원회** 신청 |
| 3단계 | 법원에 차액 반환 소송 |

> 법정 상한을 초과하는 부분은 **무효**입니다. 초과 납부한 월세를 돌려받을 수 있어요.

## 전환율 확인 방법

내가 내고 있는 월세의 전환율이 얼마인지 계산해보세요.

### 전환율 역산 공식

```
전환율(%) = (월세 × 12) ÷ (전세 시세 - 현재 보증금) × 100
```

### 예시

보증금 5,000만원, 월세 80만원, 전세 시세 3억:

```
전환율 = (80만 × 12) ÷ (3억 - 5,000만) × 100
       = 960만 ÷ 2.5억 × 100
       = 3.84%
```

법정 상한(6.25%) 이내이니 적법합니다.

### 또 다른 예시

보증금 3,000만원, 월세 120만원, 전세 시세 2.5억:

```
전환율 = (120만 × 12) ÷ (2.5억 - 3,000만) × 100
       = 1,440만 ÷ 2.2억 × 100
       = 6.55%
```

법정 상한(6.25%) **초과**! 집주인에게 조정을 요구할 수 있습니다.

## 자주 묻는 질문

### Q. 신규 계약도 법정 상한이 적용되나요?

네. 신규 계약이든 갱신 계약이든 주택 임대차라면 법정 전환율 상한이 적용됩니다.

### Q. 오피스텔도 적용되나요?

주거용 오피스텔은 적용됩니다. 주택임대차보호법의 적용을 받는 주거용 건물이면 모두 해당돼요. 상업용은 적용되지 않습니다.

### Q. 집주인이 법정 상한을 모른다고 하면?

법을 몰랐다고 해서 면책되지 않습니다. 초과 부분은 무효이고, 이미 납부한 차액은 반환 청구가 가능합니다. **주택임대차분쟁조정위원회**(☎1644-0120)에 문의하세요.

### Q. 기준금리가 바뀌면 기존 계약에도 영향이 있나요?

기존 계약의 전환율은 **계약 시점의 상한**을 기준으로 판단합니다. 기준금리가 올라서 상한이 높아져도 기존에 적법했던 전환율은 그대로 유효해요.
