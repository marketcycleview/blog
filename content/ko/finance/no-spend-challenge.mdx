"오늘 하루 0원 쓰기!"

SNS에서 유행하는 무지출 챌린지. 근데 현실적으로 가능한 걸까요?

**가능해요. 단, 규칙을 제대로 정해야 합니다.**

"고정비 제외, 변동비 0원"이 진짜 무지출이에요. 월세 내는 날은 무지출 실패? 아니에요.

## 무지출 챌린지란?

### 기본 규칙

```
무지출 = 변동비 0원인 날

포함 (무지출 인정):
- 고정비 (월세, 공과금, 보험) → 자동이체
- 교통비 (출퇴근) → 필수 지출

제외 (이것만 안 쓰면 무지출):
- 외식/배달
- 카페/간식
- 쇼핑
- 온라인 구매
- 편의점
```

### 단계별 난이도

| 단계 | 목표 | 난이도 |
|------|------|--------|
| 입문 | 주 2일 무지출 | 쉬움 |
| 중급 | 주 3일 무지출 | 보통 |
| 상급 | 주 4~5일 무지출 | 어려움 |
| 하드코어 | 1주일 무지출 | 매우 어려움 |

## 무지출 챌린지 효과

### 월 절약 효과

| 무지출 일수 | 하루 평균 절약 | 월 절약 |
|-----------|-------------|--------|
| 주 2일 (월 8일) | 15,000원 | 12만원 |
| 주 3일 (월 12일) | 15,000원 | 18만원 |
| 주 4일 (월 16일) | 15,000원 | 24만원 |

### 절약 말고 진짜 효과

| 효과 | 설명 |
|------|------|
| 소비 습관 인식 | "내가 이렇게 많이 쓰고 있었구나" |
| 충동구매 감소 | 사기 전에 한 번 더 생각 |
| 자취 습관 형성 | 배달 대신 집밥 |
| 성취감 | 하루 버티면 자신감 |

## 1주일 무지출 플랜

### 사전 준비 (챌린지 전날)

- [ ] 일주일치 식재료 장보기 (3~4만원)
- [ ] 반찬 2~3가지 만들어두기
- [ ] 간식 미리 사두기
- [ ] 카드/배달앱 잠금

### Day 1~2: 적응기

```
아침: 집밥 (시리얼, 토스트, 계란)
점심: 도시락 or 구내식당
저녁: 집밥 (미리 만든 반찬)
간식: 미리 산 것
```

**팁**: 첫 2일이 가장 힘들어요. 배달앱 알림 끄세요.

### Day 3~4: 고비

```
"카페 가고 싶다..." → 집에서 커피 내려마시기
"치킨 먹고 싶다..." → 냉동 치킨 에어프라이어
"심심하다..." → 무료 콘텐츠 (유튜브, 도서관)
```

### Day 5~7: 습관화

```
이쯤 되면 익숙해져요.
"의외로 할 만한데?"
"배달 안 시켜도 되네?"
```

## 실패 안 하는 5가지 팁

### 1. 완벽주의 버리기

하루 실패해도 챌린지 끝이 아니에요.

```
❌ "오늘 편의점 갔으니 챌린지 실패"
✅ "오늘은 실패, 내일 다시!"
```

### 2. 식재료 미리 준비

무지출 실패 원인 1위 = **먹을 게 없어서 배달.**

```
일요일에 장보기 + 밀프렙 = 평일 무지출 성공률 급상승
```

### 3. SNS/쇼핑앱 멀리하기

보면 사고 싶어져요.

```
인스타 → 쇼핑 광고 → "이거 사고 싶다" → 구매
→ 앱 알림 끄기, 홈화면에서 치우기
```

### 4. 무료 대안 준비

| 유혹 | 무료 대안 |
|------|---------|
| 카페 | 집 커피 |
| 영화관 | 넷플릭스 (이미 구독 중이면) |
| 운동 | 홈트, 런닝 |
| 모임 | 집에서 모임, 공원 산책 |

### 5. 기록하기

무지출 일수를 기록하면 동기부여가 돼요.

```
1월: 무지출 12일 → 18만원 절약
2월: 무지출 15일 → 22.5만원 절약
```

## 무지출 vs 저지출

### 무지출이 힘들면?

**저지출 챌린지**도 좋아요.

| 구분 | 규칙 |
|------|------|
| 무지출 | 변동비 0원 |
| 저지출 | 변동비 5,000원 이하 |
| 반지출 | 변동비 10,000원 이하 |

**처음엔 저지출부터 시작해서 점차 무지출로.**

## 무지출 챌린지 주의사항

### 이것만은 조심하세요

| 주의 | 이유 |
|------|------|
| 건강 식사 포기 X | 돈 아끼려다 건강 잃으면 손해 |
| 사회생활 거부 X | 모든 모임 거절은 관계 악화 |
| 극단적 절약 X | 반동으로 폭소비 가능 |
| 장기간 연속 X | 주 2~3일이 적정 |

**무지출은 "습관 리셋"이지, "생존 모드"가 아니에요.**

## 자주 묻는 질문

### Q. 출퇴근 교통비도 지출인가요?

**아니요, 고정비/필수 지출이에요.** 무지출에서 제외하세요.

### Q. 구내식당은 무지출인가요?

**규칙에 따라 다르지만**, 구내식당은 무지출로 인정하는 게 현실적이에요.

### Q. 얼마나 해야 효과 있나요?

**최소 2주.** 2주 하면 소비 패턴이 보여요. 1개월 하면 습관이 바뀌어요.

### Q. 무지출 하면 스트레스 안 받나요?

**처음엔 받아요.** 그래서 주 2~3일부터 시작하세요. 매일 하면 번아웃 와요.
