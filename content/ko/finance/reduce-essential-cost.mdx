"아끼려고 하는데 돈이 안 모여요."

카페 지출 줄이고, 택시 안 타고, 배달 줄였는데... 왜 저축이 안 늘까요?

**고정비를 안 건드렸기 때문이에요.**

커피 한 잔 아끼면 5천원. 근데 통신비 바꾸면 **월 4만원**, 보험 정리하면 **월 5만원** 아껴요.

진짜 효과 있는 건 **기초생활비(고정비) 줄이기**입니다.

## 왜 고정비부터 줄여야 하나?

### 고정비 vs 변동비

| 구분 | 예시 | 특징 |
|------|------|------|
| **고정비** | 월세, 통신비, 보험료 | 매달 자동으로 나감 |
| **변동비** | 외식, 쇼핑, 택시 | 선택에 따라 달라짐 |

### 절약 효과 비교

| 방법 | 노력 | 월 절약 | 연 절약 |
|------|------|--------|--------|
| 커피 줄이기 | 매일 참기 | 3~5만원 | 36~60만원 |
| 통신비 변경 | 1회 작업 | 3~5만원 | 36~60만원 |
| 보험 정리 | 1회 작업 | 3~10만원 | 36~120만원 |
| 월세 이사 | 1회 작업 | 10~30만원 | 120~360만원 |

**핵심**: 고정비는 **한 번 줄이면 계속 효과**가 있어요. 커피는 매일 참아야 하고요.

## 1. 통신비 최적화 (효과: 월 3~5만원)

### 알뜰폰으로 바꾸기

| 요금제 | 월 요금 | 데이터 |
|--------|--------|--------|
| 대형 통신사 | 6~8만원 | 무제한 |
| 알뜰폰 | 2~3만원 | 무제한 |
| **절약** | **4~5만원** | 같음 |

**연 48~60만원 절약!**

### 추천 알뜰폰

- 통신 품질 좋은 곳: 헬로모바일, 리브모바일 (SKT 망)
- 가성비: 유플러스알뜰모바일, 프리티 (LGU+ 망)

### 인터넷 요금

| Before | After | 절약 |
|--------|-------|------|
| 인터넷 + TV 결합 | 인터넷만 | 월 1~2만원 |
| 기가 인터넷 | 500M | 월 5천원 |

**TV 진짜 보세요?** 안 보면 해지하세요.

## 2. 보험 리모델링 (효과: 월 3~10만원)

### 20~30대 필수 보험

| 보험 | 필요성 | 권장 보험료 |
|------|--------|-----------|
| 실비보험 | 필수 | 3~5만원 |
| 자동차보험 (있다면) | 필수 | 5~10만원 |
| 암/질병보험 | 선택 | 2~3만원 |
| 종신보험 | 불필요 | 해지 고려 |
| 저축성 보험 | 불필요 | 해지 고려 |

**월 15만원 넘으면 과한 거예요.**

### 정리 방법

1. **보험다모아** (금융감독원) 접속
2. 가입 보험 조회
3. 중복/불필요 보험 확인
4. 해지 또는 감액

### 흔한 불필요 보험

| 보험 | 왜 불필요? |
|------|-----------|
| 저축성 보험 | 적금보다 수익 낮음 |
| 종신보험 (미혼) | 부양가족 없으면 불필요 |
| 2개 이상 실비 | 중복 보장 안 됨 |
| 과도한 암보험 | 실비로 대부분 커버 |

---

## 3. 구독 서비스 정리 (효과: 월 3~10만원)

### 구독 서비스 점검

흔히 있는 구독:

| 서비스 | 월 요금 |
|--------|--------|
| 넷플릭스 | 17,000원 |
| 유튜브 프리미엄 | 14,900원 |
| 스포티파이/멜론 | 10,900원 |
| 아마존 프라임 | 5,900원 |
| 앱 구독 | 5,000~15,000원 |
| **합계** | **5~7만원** |

### 정리 원칙

```
1개만 남기고 해지
```

- 영상: 넷플릭스 OR 유튜브 (하나만)
- 음악: 유튜브 프리미엄이면 뮤직도 됨
- 앱: 진짜 매일 쓰는 것만

### 확인 방법

- 아이폰: 설정 → Apple ID → 구독
- 안드로이드: Play 스토어 → 결제 및 정기 결제
- 카드사 앱: 정기결제 내역 조회

## 4. 공과금 절약 (효과: 월 2~5만원)

### 전기요금

| 방법 | 절약 효과 |
|------|---------|
| 대기전력 차단 | 월 5~10% |
| LED 교체 | 월 3~5천원 |
| 에어컨 26도 설정 | 월 1~2만원 |
| 복지 할인 신청 | 월 1~3만원 |

### 가스요금

| 방법 | 절약 효과 |
|------|---------|
| 보일러 예약 난방 | 월 1~2만원 |
| 샤워 시간 줄이기 | 월 5천원 |

### 수도요금

- 일반 가정 기준 월 1~2만원이라 절약 효과 적음
- 수도 누수 점검 정도만

### 복지 할인 (해당되면)

| 대상 | 할인 |
|------|------|
| 기초생활수급자 | 전기/가스 30~50% |
| 차상위계층 | 전기/가스 20~30% |
| 다자녀 가구 | 전기 30% |
| 장애인 | 전기/가스 30% |

## 5. 교통비 줄이기 (효과: 월 2~5만원)

### 대중교통 최적화

| 방법 | 월 절약 |
|------|--------|
| 정기권 사용 | 1~2만원 |
| 자전거/킥보드 병행 | 2~3만원 |
| 카드 교통비 할인 | 5천~1만원 |

### 자차 유지비 줄이기

| 항목 | 절약 방법 | 효과 |
|------|---------|------|
| 보험료 | 다이렉트 보험 | 10~20% |
| 주유비 | 주유 앱 할인 | 리터당 50~100원 |
| 주차비 | 월 주차 계약 | 30~50% |
| 세차 | 셀프세차 | 1만원+ |

### 차량 공유 고려

"가끔 타는데 유지비 아깝다" → 쏘카/그린카

| 비교 | 자차 | 카셰어링 |
|------|------|---------|
| 월 유지비 | 30~50만원 | 필요할 때만 |
| 월 1~2회 이용 | 비효율 | 효율적 |
| 주말만 이용 | 비효율 | 효율적 |

## 6. 식비 구조 바꾸기 (효과: 월 10~20만원)

### 식비 패턴별 비용

| 패턴 | 월 비용 (1인) |
|------|-------------|
| 매끼 배달/외식 | 90~120만원 |
| 점심 외식 + 저녁 배달 | 60~80만원 |
| 점심 외식 + 저녁 자취 | 40~55만원 |
| 대부분 자취 | 25~35만원 |

### 식비 줄이는 법

**1. 배달 횟수 제한**
- 주 3회 → 주 1회
- 월 절약: 8~12만원

**2. 점심 구내식당 활용**
- 외식 1만원 vs 구내식당 5천원
- 월 절약: 10만원

**3. 장보기 주 1회**
- 마트 대신 시장/온라인
- 주 3~4만원이면 충분

**4. 밀프렙**
- 주말에 반찬 만들어두기
- 평일 자취 쉬워짐

## 7. 주거비 최적화 (효과: 월 10~30만원)

### 월세 낮추는 방법

| 방법 | 절약 효과 |
|------|---------|
| 역세권 5분 → 15분 | 월 10~15만원 |
| 신축 → 구축 | 월 10~20만원 |
| 서울 → 경기 | 월 15~30만원 |
| 룸메이트 | 월 20~30만원 |

### 전세 vs 월세

| 상황 | 추천 |
|------|------|
| 전세 대출 이자 < 월세 | 전세 |
| 목돈 없음 | 월세 |
| 1~2년 거주 | 월세 |
| 3년 이상 거주 | 전세 |

### 주거급여 확인

중위소득 48% 이하면 **주거급여** 받을 수 있어요.

| 가구원 | 중위소득 48% | 지원 금액 (서울) |
|--------|-------------|----------------|
| 1인 | 1,125,644원 | 최대 34만원 |
| 2인 | 1,852,396원 | 최대 38만원 |

## 8. 은행 수수료 없애기 (효과: 월 1~2만원)

### 이체 수수료

| 방법 | 효과 |
|------|------|
| 인터넷뱅킹 사용 | 무료 |
| 주거래 우대 등록 | 무료 |
| 인터넷은행 사용 | 무료 |

### ATM 수수료

- 타행 ATM: 건당 500~900원
- 해결: 카카오뱅크/토스뱅크 (ATM 무료)

### 카드 연회비

- 안 쓰는 카드 해지
- 연회비 5만원 이상 카드 = 혜택 진짜 쓰는지 확인

---

## 9. 세금/공과금 자동납부 할인 (효과: 월 5천~1만원)

### 자동납부 할인

| 항목 | 할인 |
|------|------|
| 전기요금 | 1% |
| 가스요금 | 0.5~1% |
| 자동차세 | 연납 시 4.5% |
| 주민세 | 자동이체 시 할인 |

### 연납 할인

- 자동차세: 1월 연납 시 약 4.5% 할인
- 재산세: 7월, 9월 분납 가능

---

## 10. 정부/지자체 지원 활용 (효과: 월 0~20만원)

### 해당되면 신청하세요

| 지원 | 대상 | 효과 |
|------|------|------|
| 전기요금 할인 | 저소득층, 장애인 | 월 1~5만원 |
| 통신비 감면 | 저소득층, 장애인 | 월 1~2만원 |
| 주거급여 | 중위소득 48% 이하 | 월 최대 34만원 |
| 가스요금 할인 | 저소득층 | 월 1~3만원 |

### 확인 방법

- **복지로** (www.bokjiro.go.kr)
- **정부24** (www.gov.kr)
- **주민센터** 방문 상담

## 고정비 점검 체크리스트

**통신비**
- [ ] 알뜰폰으로 바꿨다
- [ ] 안 보는 TV 해지했다
- [ ] 인터넷 속도 적정하다

**보험료**
- [ ] 보험다모아에서 조회했다
- [ ] 중복 보험 없다
- [ ] 월 15만원 이하다

**구독 서비스**
- [ ] 전체 구독 목록 확인했다
- [ ] 안 쓰는 구독 해지했다
- [ ] 영상/음악 각 1개씩만

**주거비**
- [ ] 소득의 25% 이하다
- [ ] 주거급여 대상인지 확인했다

**공과금**
- [ ] 자동납부 설정했다
- [ ] 복지 할인 대상인지 확인했다
