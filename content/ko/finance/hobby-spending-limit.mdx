"헬스장은 투자고, 골프는 사치인가요?"

그건 **소득에 따라 달라요.**

월급 300만원에 월 50만원 골프 = 소득의 17% → 과소비
월급 700만원에 월 50만원 골프 = 소득의 7% → 여유비

핵심은 **금액이 아니라 비율**이에요. 소득의 **3~5%가 적정 취미비**입니다.

## 소득별 적정 취미비

| 월 실수령 | 적정 (3%) | 최대 (5%) | 예시 |
|---------|---------|---------|------|
| 170만원 | 5.1만원 | 8.5만원 | 헬스장 |
| 250만원 | 7.5만원 | 12.5만원 | 헬스장 + 취미 1개 |
| 340만원 | 10.2만원 | 17만원 | 운동 + 취미 |
| 410만원 | 12.3만원 | 20.5만원 | 운동 + 취미 여유 |
| 550만원 | 16.5만원 | 27.5만원 | 다양한 취미 가능 |

## 투자인 취미 vs 낭비인 취미

### 투자로 볼 수 있는 취미

| 취미 | 투자 효과 |
|------|---------|
| 운동 (헬스/런닝) | 건강 → 의료비 절감, 생산성 향상 |
| 독서/강의 | 지식 → 커리어 성장 |
| 외국어 | 스킬 → 연봉 상승 가능 |
| 네트워킹 | 인맥 → 기회 |

### 순수 여가 (OK, 적정선 안에서)

| 취미 | 기준 |
|------|------|
| 게임 | 월 3만원 이하 |
| 영화/공연 | 월 5만원 이하 |
| 카페 | 소득의 2~3% |
| 여행 | 연간 예산 세우기 |

### 주의해야 할 취미

| 취미 | 위험 |
|------|------|
| 골프 (소득 대비 과한 경우) | 장비 + 라운딩 + 모임 → 폭발적 지출 |
| 수집 (피규어/레고/카드) | 끝이 없음 |
| 도박성 (경마/로또 과다) | 중독 위험 |

## 취미비 관리 방법

### 1. 월 예산 정하기

```
여유비의 30~40%를 취미에 배정

월급 300만원 → 여유비 60만원 → 취미비 18~24만원
```

### 2. 연간 예산으로 관리

비정기 취미 (여행, 장비 구매)는 월이 아닌 **연간 예산**으로:

```
연간 취미 예산: 150만원
- 헬스장: 월 8만원 × 12 = 96만원
- 여행: 연 2회 = 40만원
- 기타: 14만원
```

### 3. "1 in 1 out" 규칙

새 취미 시작하면 기존 취미 하나 줄이기.

```
골프 시작 → 헬스 해지 (X, 건강은 유지)
골프 시작 → 구독 서비스 줄이기 (O)
```

## 자주 묻는 질문

### Q. 헬스장은 낭비인가요?

**아니요, 건강 투자예요.** 월 5~10만원으로 건강 유지하면 의료비 수백만원 아낄 수 있어요.

### Q. PT는 사치인가요?

**소득에 따라 다릅니다.**

| 월급 | PT 주 2회 (40만원) | 비율 | 판정 |
|------|------------------|------|------|
| 250만원 | 16% | 과소비 |
| 400만원 | 10% | 경계 |
| 600만원 | 6.7% | 여유비 |

### Q. 취미 없이 아끼기만 해야 하나요?

**절대 아니에요.** 여유비 0%는 번아웃으로 가는 길이에요. 적정선 안에서 즐기는 게 오래 갑니다.
