연봉 협상을 안 하면, 회사가 제시한 금액이 곧 내 연봉이 됩니다.

"신입인데 협상해도 되나?", "얼마를 불러야 하지?" 하고 망설이다 그냥 사인하는 분이 많아요. 그런데 한 번 정해진 연봉은 다음 해 인상률의 기준이 되기 때문에, 초반에 협상을 안 하면 격차가 계속 벌어집니다.

신입과 경력직 상황별로 나눠서 정리했습니다.

---

## 연봉 협상, 왜 해야 하나

### 복리 효과

| 시작 연봉 | 5년 후 (매년 5% 인상) |
|----------|---------------------|
| 3,000만원 | 3,829만원 |
| 3,200만원 (200만원 높게 시작) | 4,084만원 |
| **5년 누적 차이** | **약 1,100만원** |

시작 연봉 200만원 차이가 5년이면 1,100만원 이상 벌어집니다. 협상은 입사 시점에 하는 게 가장 효과가 커요.

---

## 신입 연봉 협상

### 협상 가능한 타이밍

| 시점 | 협상 가능? |
|------|----------|
| 서류 지원 시 | ❌ (너무 이름) |
| 면접 중 "희망 연봉" 질문 | △ (범위로 답변) |
| **최종 합격 후 ~ 입사 전** | ✅ (가장 좋은 타이밍) |
| 입사 후 | ❌ (이미 계약 완료) |

최종 합격 통보를 받은 후, 연봉 제안서(오퍼)를 받았을 때가 최적입니다. 회사가 채용을 결정한 상태라 협상 여지가 있어요.

### 신입이 쓸 수 있는 근거

| 근거 | 예시 |
|------|------|
| 동종업계 신입 평균 연봉 | "같은 업종 평균이 3,200만원입니다" |
| 자격증·어학 | "회계사/변호사/정보처리기사 보유" |
| 인턴 경험 | "같은 회사 인턴 6개월 근무" |
| 동시 합격 오퍼 | "다른 곳에서 3,300만원 제안받았습니다" |

### 신입 협상 멘트 예시

```
"합격 제안 감사합니다. 제시하신 연봉에 대해 여쭤봐도 될까요?
동종업계 신입 평균과 비교했을 때, OO만원 정도로 조정이 가능할지
검토 부탁드려도 될까요?"
```

**주의**: 신입은 100~300만원 범위가 현실적이에요. 1,000만원 올려달라고 하면 역효과가 날 수 있습니다.

## 경력직 연봉 협상

### 이직 시 적정 인상 폭

| 이직 상황 | 일반적 인상 폭 |
|----------|--------------|
| 동종업계 수평 이동 | 10~20% |
| 더 큰 회사로 이직 | 15~30% |
| 스카우트/헤드헌팅 | 20~40% |
| 업종 전환 | 0~10% (경력 인정 범위에 따라) |

현재 연봉의 **15~25% 인상**이 가장 흔한 범위입니다. 그 이상 요구하려면 강력한 근거가 필요해요.

### 경력직이 쓸 수 있는 근거

| 근거 | 활용법 |
|------|--------|
| 현재 연봉 | "현재 OO만원 받고 있습니다" |
| 경력 연차 | "해당 분야 N년 경력입니다" |
| 성과 실적 | "매출 30% 성장을 이끌었습니다" |
| 시장 시세 | "잡코리아/사람인 통계 기준 평균 OO만원" |
| 동시 오퍼 | "다른 곳에서 OO만원을 제시받았습니다" |
| 전문 자격 | "관련 자격증/특허 보유" |

### 경력직 협상 멘트 예시

```
"좋은 기회를 주셔서 감사합니다. 제 경력과 성과를 고려했을 때,
OO만원 수준이 적절하다고 생각합니다.
현재 직장에서 [구체적 성과]를 담당했고,
이 경험을 바탕으로 귀사에서도 기여할 수 있다고 확신합니다."
```

## 재직 중 연봉 인상 협상

이직이 아니라 현 직장에서 연봉을 올리고 싶을 때입니다.

### 타이밍

| 시점 | 효과 |
|------|------|
| **연봉 협상 시즌** (12~2월) | 가장 자연스러움 |
| 성과 평가 직후 | 근거가 명확할 때 |
| 대형 프로젝트 완료 후 | 기여도 입증 가능 |
| 승진 시 | 직급에 맞는 연봉 요구 |

### 준비해야 할 것

| 항목 | 내용 |
|------|------|
| 성과 정리 | 숫자로 표현 (매출, 비용 절감, 고객 증가 등) |
| 시장 시세 | 동종업계 같은 직급 평균 연봉 |
| 추가 역할 | 원래 JD 외에 맡고 있는 업무 |
| 희망 금액 | 구체적 숫자 (범위보다 한 금액이 낫다) |

### 멘트 예시

```
"올해 [구체적 성과]를 달성했고, 팀 내에서 [추가 역할]도 맡고 있습니다.
업계 평균과 제 기여도를 고려해서, 연봉 OO만원으로 조정을 요청드립니다."
```

## 협상에서 피해야 할 실수

| 실수 | 왜 문제인가 |
|------|-----------|
| "생활비가 부족해서요" | 개인 사정은 협상 근거가 아님 |
| "동기가 더 많이 받아서요" | 비교 대상이 명확하지 않음 |
| "얼마든 괜찮아요" | 협상 포기 = 불리한 조건 수락 |
| 첫 제안에 바로 수락 | 회사도 여유를 두고 제안함 |
| 허풍 오퍼 (거짓 제안) | 확인되면 신뢰 상실 |
| 이메일/문자로 협상 | 대면이나 전화가 효과적 |

## 연봉 외 협상 가능한 항목

연봉이 더 이상 안 올라갈 때, 다른 조건을 협상할 수 있습니다.

| 항목 | 예시 |
|------|------|
| 사이닝 보너스 | 입사 시 일시금 (100~500만원) |
| 성과급 보장 | "첫 해 성과급 최소 OO% 보장" |
| 연차 추가 | 법정 연차 외 추가 유급휴가 |
| 재택근무 | 주 1~2회 재택 |
| 교육비 지원 | 학원, 자격증, 대학원 |
| 스톡옵션 | 스타트업의 경우 |
| 직급 | 경력 인정 직급 |
| 수습 기간 면제 | 경력직의 경우 |

> 연봉 200만원 인상이 안 되면, 사이닝 보너스 200만원을 요청하는 것도 방법입니다. 회사 입장에서는 고정비가 아니라 수용하기 쉬워요.

## 자주 묻는 질문

### Q. 신입인데 연봉 협상하면 인상 안 좋은가요?

합리적인 범위에서 근거를 갖고 요청하면 문제없습니다. 오히려 회사 입장에서도 자기 가치를 아는 지원자를 긍정적으로 볼 수 있어요. 다만 무리한 금액이나 태도는 역효과가 납니다.

### Q. "희망 연봉"을 물어보면 뭐라고 해야 하나요?

구체적 숫자보다 **범위**로 답하세요. "업계 평균인 3,200~3,500만원 수준을 생각하고 있습니다"가 "3,500만원이요"보다 안전합니다.

### Q. 현재 연봉을 솔직하게 말해야 하나요?

경력직이라면 현재 연봉은 협상의 출발점이 됩니다. 과장하면 입사 후 원천징수 영수증으로 확인될 수 있어요. 솔직하되, "현재 OO이지만 이직 시 OO 수준을 기대합니다"로 방향을 잡으세요.

### Q. 협상이 결렬되면 합격이 취소되나요?

일반적으로 취소되지 않습니다. 회사가 수용 불가하면 원래 제안으로 돌아갈 뿐이에요. 다만 "이 금액 아니면 안 갑니다" 식의 최후통첩은 피하세요.
