저소득층 기저귀·조제분유 지원, 영아 양육비 부담을 덜어줍니다.

만 2세 미만 영아를 키우는 저소득 가정에 **기저귀 월 9만원**, **조제분유 월 11만원**을 지원합니다. 국민행복카드로 편리하게 사용할 수 있어요.

## 기저귀·조제분유 지원 핵심 정보

| 항목 | 내용 |
|------|------|
| 대상 | 만 2세 미만 영아가 있는 저소득 가정 |
| 기저귀 | **월 9만원** |
| 조제분유 | **월 11만원** |
| 합계 | **월 최대 20만원** |
| 지원기간 | 최대 24개월 |
| 신청 | 보건소, 주민센터, 복지로 |

---

## 지원 대상

### 기저귀 지원

만 2세 미만(0~24개월) 영아가 있는 다음 가정:

| 대상 | 소득 기준 |
|------|----------|
| **기초생활수급자** | 생계·의료·주거·교육급여 |
| **차상위계층** | 차상위 확인 |
| **한부모가족** | 한부모가족지원법 대상 |
| **장애인 가구** | 중위소득 80% 이하 |
| **다자녀 가구** (2인 이상) | 중위소득 80% 이하 |

### 조제분유 지원

기저귀 지원 대상 중 **아래 조건**에 해당해야 합니다:

| 조건 | 설명 |
|------|------|
| **산모 사망** | 가족관계증명서로 확인 |
| **산모 질병** | 의사 진단서/소견서 필요 |
| **한부모 (부자가정/조손가정)** | 아버지 또는 조부모가 양육 |
| **시설 입소 아동** | 아동복지시설, 공동생활가정 등 |
| **입양 아동** | 입양 가정의 아동 |
| **가정위탁 아동** | 가정위탁 보호 아동 |

> **주의**: 조제분유는 **모유수유가 불가능한 경우**에만 지원됩니다.

## 지원 금액

| 지원 유형 | 월 지원금액 |
|----------|-----------|
| 기저귀만 | **9만원** |
| 조제분유만 | **11만원** |
| 기저귀 + 조제분유 | **20만원** |

### 바우처 지급 방식

- **3개월 단위**로 지급
- 기저귀: 27만원 (3개월분)
- 기저귀+조제분유: 60만원 (3개월분)

### 사용 방법

기저귀+조제분유 지원을 받으면 **통합 바우처**로 지급됩니다. 기저귀와 조제분유를 **구분 없이** 자유롭게 구매할 수 있어요.

---

## 지원 기간

- **최대 24개월** (영아 출생~만 2세)
- 출생일로부터 **60일 이내** 신청 시 24개월 전액 지원
- 60일 이후 신청 시 신청일부터 남은 기간만 지원

---

## 신청 방법

### 신청 기간

영아 출생 후 ~ **만 2세가 되기 전날**까지

### 방문 신청

**신청 장소**: 영아 주소지 관할 **보건소** 또는 **읍면동 주민센터**

**준비 서류**:
- 저소득층 기저귀·조제분유 지원 신청서
- 신분증
- 건강보험증 사본
- 건강보험료 납부확인서 (소득 확인용)
- 주민등록등본, 가족관계증명서

**조제분유 추가 서류**:
- 산모 사망: 가족관계증명서
- 산모 질병: 의사 진단서/소견서
- 시설 입소: 입소 확인서
- 입양: 입양 확인서

## 사용 방법

### 국민행복카드

지원금은 **국민행복카드**에 바우처로 충전됩니다.

### 구매 가능 장소

- 대형마트 (이마트, 홈플러스, 롯데마트 등)
- 편의점
- 베이비용품 전문점
- 온라인 쇼핑몰 (바우처 결제 가능 업체)

### 구매 가능 품목

| 품목 | 예시 |
|------|------|
| **기저귀** | 일회용 기저귀, 천 기저귀 |
| **조제분유** | 분유, 특수 조제분유 |

> **주의**: 물티슈, 유아 간식 등은 **구매 불가**입니다. 기저귀와 조제분유만 가능해요.

## 잔액 확인 및 이월

### 잔액 확인

- 국민행복카드 카드사 앱/홈페이지
- 사회서비스 전자바우처 홈페이지

### 이월

사용하지 않은 바우처는 **지원 종료일까지 이월**되어 계속 사용할 수 있습니다.

## 다른 지원과 중복

### 중복 불가

- 영양플러스사업 (조제분유) + 조제분유 지원 ❌
- 선천성대사이상 환아관리사업 (조제분유) + 조제분유 지원 ❌

### 중복 가능

- 부모급여 + 기저귀·조제분유 지원 ⭕
- 아동수당 + 기저귀·조제분유 지원 ⭕
- 첫만남이용권 + 기저귀·조제분유 지원 ⭕

## 자주 묻는 질문

**Q. 다자녀인데 모든 아이에게 지원받을 수 있나요?**

만 2세 미만 영아만 지원 대상입니다. 다만, 둘째 출생 시 첫째가 24개월 미만이면 **첫째도 함께 지원** 받을 수 있습니다.

**Q. 분유 대신 기저귀만 더 살 수 있나요?**

기저귀+조제분유 통합 지원을 받으면 구분 없이 자유롭게 사용 가능합니다. 분유 대신 기저귀만 사도 됩니다.

**Q. 국민행복카드가 없으면 어떻게 하나요?**

보건소나 주민센터에서 신청 시 국민행복카드 발급도 함께 안내받을 수 있습니다.

**Q. 지원 중간에 자격을 잃으면 어떻게 되나요?**

자격 상실 시 그 다음 달부터 지원이 중단됩니다. 이미 충전된 바우처는 지원 종료일까지 사용 가능합니다.
