## 소득 무관 혜택 (4인가구 해당)

소득과 관계없이 받을 수 있는 혜택도 있습니다.

| 정책 | 대상 | 지원 |
|------|------|------|
| 아동수당 | 만 8세 미만 | 월 10만원/인 |
| 부모급여 | 0~1세 | 0세 100만 / 1세 50만원 |
| 어린이집 무상보육 | 0~5세 | 보육료 전액 |
| 아이돌봄서비스 | 150% 이하 | 시간당 돌봄 |
| 초등돌봄교실 | 초등 1~2학년 | 방과후 돌봄 |

4인가구에 만 8세 미만 자녀 2명이면 아동수당만 **월 20만원**입니다.

## 4인가구 소득인정액 예시

### 외벌이, 월급 350만원, 서울 전세 2억

| 항목 | 금액 |
|------|------|
| 근로소득 | 350만원 |
| 근로소득공제 (30%) | -105만원 |
| 소득평가액 | 245만원 |
| 재산 (전세 2억) | 2억 |
| 기본공제 (서울) | -9,900만원 |
| 재산환산 (1.01억 × 4.17%) | 421.2만원 |
| **소득인정액** | **약 666만원** |

```
중위소득 비율: 666만 ÷ 649만 × 100 = 약 103%
→ 기초급여 해당 안 됨
→ 아동수당, 보육료 등 소득 무관 혜택만 가능
```

### 외벌이, 월급 200만원, 지방 월세 (보증금 3천만원)

| 항목 | 금액 |
|------|------|
| 근로소득 | 200만원 |
| 근로소득공제 (30%) | -60만원 |
| 소득평가액 | 140만원 |
| 재산 (보증금 3천만원) | 3,000만원 |
| 기본공제 (지방) | -5,300만원 |
| 재산환산 | 0원 (공제 이내) |
| **소득인정액** | **약 140만원** |

```
중위소득 비율: 140만 ÷ 649만 × 100 = 약 21.6%
→ 생계급여(32%) 이하 → 4대 급여 전부 수급
→ 생계급여: 208만 - 140만 = 약 68만원 지급
```

## 자주 묻는 질문

### Q. 4인가구 기준이 가장 많이 쓰이는 이유는?

정부가 복지 기준을 발표할 때 대표적으로 4인가구를 기준으로 제시합니다. 평균적인 가구 규모로 활용되기 때문이에요. 하지만 실제 수급 판정은 **본인 가구원수** 기준입니다.

### Q. 자녀가 대학생이어도 가구원에 포함되나요?

미혼이고 부모와 함께 생계를 유지하면 포함됩니다. 대학 때문에 다른 지역에 거주해도 **독립 가구임을 증빙하지 않으면** 부모 가구에 포함돼요.

### Q. 맞벌이 4인가구도 복지 받을 수 있나요?

가능합니다. 4인가구 100% 기준이 649만원이고, 맞벌이 합산소득에서 각각 30% 공제가 적용되므로 중간소득 맞벌이도 80% 이하에 해당할 수 있어요. 특히 재산이 적으면 가능성이 높습니다.

### Q. 다자녀 혜택은 별도인가요?

자녀 3명 이상이면 **다자녀 특별공급**(주택), 자동차 취득세 감면, 대학등록금 지원 등 별도 혜택이 있습니다. 중위소득 기준과는 별개예요.
