실업급여 수급 자격은 **4가지 조건**을 모두 충족해야 합니다. 하나라도 빠지면 받을 수 없어요. 내가 해당되는지 하나씩 체크해보세요.

---

## 수급 자격 4가지 조건

| # | 조건 | 기준 |
|---|------|------|
| ① | 고용보험 가입 기간 | 퇴직 전 18개월 중 **180일 이상** |
| ② | 퇴직 사유 | **비자발적** (해고, 권고사직, 계약만료 등) |
| ③ | 근로 능력·구직 의사 | 일할 수 있고 적극적으로 **구직 활동** |
| ④ | 신청 기한 | 퇴직일 다음 날부터 **12개월 이내** |

## ① 고용보험 가입 180일 이상

### 어떻게 계산하나?

| 항목 | 내용 |
|------|------|
| 기준 기간 | 퇴직 전 **18개월** |
| 필요 일수 | **180일** (약 6개월) |
| 계산 방법 | 유급일수(월급 받은 날) 기준 |
| 주 5일제 | 월 약 22일 × 8개월 = 약 176일 → **부족** |
| 주 5일제 | 월 약 22일 × 9개월 = 약 198일 → **충족** |

> **주의**: 달력상 6개월이 아니라 **유급 180일**입니다. 무급휴가, 결근일은 제외되므로 실제로는 9개월 이상 근무해야 안전합니다.

### 여러 직장 합산 가능

| 상황 | 합산 여부 |
|------|----------|
| A회사 100일 + B회사 80일 | **가능** (합계 180일) |
| 이전 직장에서 실업급여 받음 | 그 이후 기간만 합산 |
| 1개월 공백 후 재취업 | **가능** (18개월 이내면) |

### 확인 방법

고용24 → 마이페이지 → **피보험 자격 이력** 조회하면 가입 기간을 확인할 수 있습니다.

## ② 비자발적 퇴사

### 인정되는 퇴직 사유

| 사유 | 인정 여부 |
|------|----------|
| 해고 | ✅ |
| 권고사직 | ✅ |
| 계약 만료 (갱신 거절) | ✅ |
| 폐업·도산 | ✅ |
| 임금 체불 (2개월+) | ✅ (자발적이어도) |
| 직장 내 괴롭힘 | ✅ (자발적이어도) |
| 건강 악화 | ✅ (진단서 필요) |
| 배우자 전근 | ✅ |
| 통근 곤란 (왕복 3시간+) | ✅ |

### 인정 안 되는 사유

| 사유 | 인정 여부 |
|------|----------|
| 단순 이직 (다른 회사 가려고) | ❌ |
| 그냥 쉬고 싶어서 | ❌ |
| 사업하려고 | ❌ |
| 직장 불만 | ❌ |

> 자발적 퇴사도 **정당한 사유**가 있으면 인정됩니다. 자세한 내용은 [자발적 퇴사 실업급여](/ko/finance/jabaljeok-toesa-silup)를 확인하세요.

## ③ 근로 능력·구직 의사

| 조건 | 설명 |
|------|------|
| 근로 능력 | 일할 수 있는 건강 상태 |
| 구직 의사 | 적극적으로 일자리를 찾고 있음 |
| 구직 활동 | 2주마다 구직활동 내역 보고 (실업 인정) |

### 구직 활동으로 인정되는 것

```
✅ 입사 지원 (온라인/오프라인)
✅ 면접 참석
✅ 직업훈련 수강
✅ 취업 박람회 참가
✅ 고용센터 상담
❌ 단순 채용공고 검색만
❌ 이력서 작성만 (지원까지 해야 함)
```

## ④ 12개월 이내 신청

| 항목 | 내용 |
|------|------|
| 시작일 | 퇴직일 다음 날 |
| 마감일 | 퇴직일 + **12개월** |
| 늦게 신청하면 | 남은 기간이 줄어듦 |

예: 지급일수 180일인데 퇴직 후 10개월에 신청하면 → 12개월 - 10개월 = 2개월(60일)만 받을 수 있음.

## 자주 묻는 질문

### Q. 고용보험 가입 기간이 180일 미만이면?

실업급여를 받을 수 없습니다. 다만 **국민취업지원제도**(구직촉진수당)는 고용보험 가입과 무관하므로 대안이 될 수 있어요.

### Q. 계약직인데 내가 갱신을 거부했다면?

본인이 갱신을 거부한 경우는 원칙적으로 자발적 퇴사입니다. 다만 근로조건이 크게 나빠진 경우 등 **정당한 사유**가 있으면 인정받을 수 있어요.

### Q. 3개월 근무 후 해고됐는데 받을 수 있나요?

고용보험 가입 기간이 180일 미만이면 받을 수 없습니다. 다만 이전 직장의 가입 기간을 **합산**할 수 있으니, 고용24에서 피보험 이력을 확인해보세요.

### Q. 프리랜서도 실업급여 받나요?

고용보험에 가입된 프리랜서(특수고용직)는 가능합니다. 2021년부터 보험설계사, 학습지교사, 배달기사, 대리운전 등도 고용보험 의무가입 대상이에요.

### Q. 수급자격 심사에 얼마나 걸리나요?

보통 **1~2주**입니다. 퇴직 사유가 명확하고 서류가 완비되면 빨리 처리돼요.
