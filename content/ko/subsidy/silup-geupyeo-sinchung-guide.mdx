실업급여 신청, 어디서 어떻게 하면 될까요?

퇴사 후 **고용24**(온라인) 또는 **고용센터**(오프라인)에서 신청합니다. 이직확인서 발급부터 첫 수급까지 약 **2~3주**가 걸려요. 단계별로 정리했습니다.

---

## 신청 절차 요약

| 단계 | 내용 | 소요 |
|------|------|------|
| 1 | 퇴사 + 이직확인서 발급 | 퇴직 후 자동 (10일 이내) |
| 2 | 고용24 구직 등록 | 당일 |
| 3 | 수급자격 인정 신청 | 1~2일 |
| 4 | 수급자격 교육 이수 (온라인) | 약 1시간 |
| 5 | 대기기간 (7일) | 7일 |
| 6 | 실업 인정 (2주마다) | 정기 반복 |
| 7 | 실업급여 입금 | 인정 후 1~2일 |

## 1단계: 이직확인서 발급

퇴사하면 회사가 **10일 이내**에 고용센터에 이직확인서를 제출해야 합니다.

| 항목 | 내용 |
|------|------|
| 발급 주체 | **회사** (사업주) |
| 제출처 | 고용센터 (온라인 전산) |
| 기한 | 퇴직 후 10일 이내 |
| 확인 방법 | 고용24에서 "피보험자격 이력" 조회 |

> **회사가 안 해주면?** 고용센터(☎1350)에 신고하세요. 이직확인서 미제출 시 사업주에게 과태료가 부과됩니다.

### 온라인 (고용24)

|------|------|

| 필요 | 공인인증서/간편인증 |

| 준비물 | 신분증, 통장사본 |
| 방문 | 거주지 관할 고용센터 |

## 3단계: 수급자격 인정 신청

구직 등록 후 **수급자격 인정 신청서**를 제출합니다.

| 필요 서류 | 비고 |
|----------|------|
| 수급자격 인정 신청서 | 고용24 또는 고용센터에서 작성 |
| 신분증 | 주민등록증, 운전면허증 |
| 통장 사본 | 실업급여 입금 계좌 |
| 이직확인서 | 회사 제출 여부 확인 |

### 심사 내용

| 확인 사항 | 기준 |
|----------|------|
| 고용보험 가입 기간 | 퇴직 전 18개월 중 180일 이상 |
| 퇴직 사유 | 비자발적 (해고, 권고사직, 계약만료 등) |
| 근로 능력 및 구직 의사 | 적극적 구직 활동 의향 |

---

## 4단계: 수급자격 교육

| 항목 | 내용 |
|------|------|
| 교육 방법 | **온라인** (고용24) 또는 고용센터 방문 |
| 소요 시간 | 약 1시간 |
| 내용 | 실업급여 제도, 구직활동 안내, 부정수급 주의 |
| 이수 시기 | 수급자격 신청 후, 첫 실업인정 전 |

---

## 5단계: 대기기간 (7일)

수급자격이 인정되면 **7일간 대기기간**을 거칩니다. 이 기간에는 실업급여가 지급되지 않아요.

> 자발적 퇴사(정당한 사유 인정)의 경우 대기기간이 **1~3개월**로 늘어날 수 있습니다.

---

## 6단계: 실업 인정 (2주마다)

| 항목 | 내용 |
|------|------|
| 주기 | **2주에 1번** |
| 방법 | 고용24 온라인 또는 고용센터 방문 |
| 신고 내용 | 구직활동 내역, 소득 발생 여부 |
| 구직활동 최소 횟수 | 1~4주차 1회 이상 |

|------|----------|

| 면접 참석 | ✅ |
| 직업훈련 수강 | ✅ |
| 취업 박람회 참가 | ✅ |
| 고용센터 상담 | ✅ |
| 단순 채용공고 검색 | ❌ (지원까지 해야 인정) |

## 7단계: 실업급여 입금

실업 인정 후 **1~2영업일** 이내에 지정 계좌로 입금됩니다.

| 항목 | 내용 |
|------|------|
| 지급 금액 | 일 64,192~77,000원 × 인정일수 |
| 지급 주기 | 2주마다 |
| 과세 | **비과세** (세금 없음) |
| 총 수급 기간 | 120~270일 (가입기간·나이별) |

## 자주 묻는 질문

### Q. 퇴사 후 언제까지 신청해야 하나요?

퇴직일 다음 날부터 **12개월 이내**에 신청해야 합니다. 늦게 신청하면 남은 기간이 12개월을 넘기면서 지급일수가 줄어들 수 있어요. 가능한 빨리 신청하세요.

### Q. 온라인으로만 신청 가능한가요?

아닙니다. 고용24(온라인)과 고용센터(오프라인) 모두 가능합니다. 첫 신청은 고용센터 방문을 추천하는데, 상담을 받으면서 절차를 진행할 수 있어서요.

### Q. 수급자격이 불인정되면?

**재심사 요청(이의 신청)**을 할 수 있습니다. 불인정 통보일로부터 90일 이내에 고용센터에 이의 신청서를 제출하세요. 증빙 서류를 추가로 제출하면 결과가 바뀔 수 있어요.

### Q. 구직활동을 안 하면 어떻게 되나요?

실업 인정이 되지 않아 해당 기간의 **실업급여가 지급되지 않습니다**. 다만 지급일수가 소멸되는 건 아니고 뒤로 밀려요. 단, 정당한 사유 없이 계속 구직활동을 하지 않으면 수급자격이 취소될 수 있습니다.

### Q. 실업급여 받는 동안 건강보험은?

수급 기간 중 **임의계속가입**을 신청하면 직장 가입자 때의 보험료로 유지할 수 있습니다. 지역가입자로 전환되면 보험료가 올라갈 수 있으니 확인하세요.
