실직 후 받을 수 있는 돈이 실업급여만 있는 건 아닙니다. **구직촉진수당**(국민취업지원제도)이라는 제도도 있어요.

둘 다 구직자를 위한 지원금이지만 조건, 금액, 기간이 완전히 다릅니다. 내 상황에 맞는 제도를 확인하세요.

---

## 한눈에 비교

| 항목 | **실업급여** | **구직촉진수당** |
|------|------------|----------------|
| 근거 법 | 고용보험법 | 구직자취업촉진법 |
| 지급 기관 | 고용보험 | 국민취업지원제도 |
| **자격 기준** | 고용보험 180일 | 중위소득 60% 이하 |
| **퇴사 사유** | 비자발적 | **무관** |
| **월 지급액** | **193~231만원** | **50만원** |
| **총 지급액** | 771~2,079만원 | **300만원** |
| 지급 기간 | 120~270일 | 6개월 |
| 과세 | 비과세 | 비과세 |
| 구직활동 | 2주마다 인정 | 매월 활동 보고 |

## 실업급여 상세

### 수급 조건

| 조건 | 기준 |
|------|------|
| 고용보험 가입 | 18개월 중 180일 이상 |
| 퇴사 사유 | **비자발적** (해고, 권고사직, 계약만료) |
| 구직 활동 | 2주마다 구직활동 보고 |
| 신청 기한 | 퇴직 후 12개월 이내 |

### 지급 금액

| 항목 | 금액 |
|------|------|
| 일 하한액 | 64,192원 (최저임금 80%) |
| 일 상한액 | 77,000원 |
| 월 하한 | 약 193만원 |
| 월 상한 | 약 231만원 |
| 소득 대체율 | 퇴직 전 평균임금의 60% |

### 지급 기간

| 가입기간 | 50세 미만 | 50세 이상 |
|---------|----------|----------|
| 1년 미만 | 120일 | 120일 |
| 1~3년 | 150일 | 180일 |
| 3~5년 | 180일 | 210일 |
| 5~10년 | 210일 | 240일 |
| 10년 이상 | 240일 | 270일 |

## 구직촉진수당 상세 (국민취업지원제도 1유형)

### 수급 조건

| 조건 | 기준 |
|------|------|
| 소득 | 중위소득 **60% 이하** |
| 재산 | **5억원 이하** |
| 나이 | 15~69세 |
| 퇴사 사유 | **무관** (자발적도 가능) |
| 고용보험 | **무관** |

### 지급 금액

| 항목 | 금액 |
|------|------|
| 월 지급액 | **50만원** |
| 총 지급 | **300만원** (6개월) |
| 취업활동비 | 추가 지원 가능 |
| 부양가족 추가 | 1인당 10만원 (최대 40만원 추가) |

부양가족이 있으면 최대 **월 90만원**까지 받을 수 있습니다.

## 상황별 추천

| 상황 | 추천 |
|------|------|
| 고용보험 가입 + 비자발적 퇴사 | **실업급여** (금액이 훨씬 큼) |
| 고용보험 미가입 + 저소득 | **구직촉진수당** |
| 자발적 퇴사 + 정당한 사유 | **실업급여** 시도 → 불인정 시 구직촉진수당 |
| 자발적 퇴사 + 저소득 | **구직촉진수당** |
| 프리랜서·일용직 (고용보험 미가입) | **구직촉진수당** |
| 졸업 후 미취업 | **구직촉진수당** (2유형) |

### 중복 수급 가능?

실업급여와 구직촉진수당은 **동시에 받을 수 없습니다**. 실업급여 수급 종료 후 구직촉진수당을 신청하는 건 가능해요.

---

## 국민취업지원제도 2유형

1유형 외에 소득 기준이 더 넓은 **2유형**도 있습니다.

| 항목 | 1유형 | 2유형 |
|------|-------|-------|
| 소득 기준 | 60% 이하 | **100% 이하** |
| 월 수당 | 50만원 | - |
| 취업활동비 | ✅ | 최대 **195.4만원** |
| 취업성공수당 | - | 최대 150만원 |

2유형은 현금 수당은 없지만, 취업 프로그램과 취업활동비를 지원받을 수 있습니다.

## 자주 묻는 질문

### Q. 실업급여가 더 많으니까 무조건 실업급여 받으면 되나요?

실업급여는 고용보험 가입 180일 + 비자발적 퇴사가 조건입니다. 이 조건을 충족하지 못하면 받을 수 없어요. 그때는 구직촉진수당이 대안이 됩니다.

### Q. 실업급여 끝나고 구직촉진수당 받을 수 있나요?

가능합니다. 실업급여 수급 종료 후에도 소득 기준(60% 이하)을 충족하면 구직촉진수당을 신청할 수 있어요.

### Q. 아르바이트를 하면 구직촉진수당이 줄어드나요?

구직촉진수당은 소득에 따라 감액되지 않습니다. 다만 취업(월 60시간 이상 근로)하면 수급이 종료돼요.

### Q. 대학 졸업 후 취업 못 했으면 뭘 신청하나요?

고용보험 이력이 없으니 실업급여는 불가합니다. **국민취업지원제도 2유형**이 적합해요. 소득 기준만 충족하면 취업활동비와 프로그램을 지원받을 수 있습니다.
