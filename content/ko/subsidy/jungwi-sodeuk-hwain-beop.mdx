내 소득이 중위소득 몇 프로인지 어떻게 확인할까요?

**3단계**면 됩니다. 가구원수 확인 → 소득인정액 계산 → 기준 중위소득과 비교. 복잡할 것 같지만 계산기를 쓰면 간단해요.

---

## 확인 3단계

| 단계 | 내용 | 방법 |
|------|------|------|
| 1단계 | **가구원수** 확인 | 주민등록등본 기준 |
| 2단계 | **소득인정액** 계산 | 소득 + 재산환산 |
| 3단계 | **비율** 확인 | 소득인정액 ÷ 기준 중위소득 × 100 |

## 2단계: 소득인정액 계산

단순히 월급만 보는 게 아닙니다. **소득인정액**이라는 공식 기준으로 판단해요.

### 소득인정액 공식

```
소득인정액 = 소득평가액 + 재산의 소득환산액
```

### 소득평가액 계산

```
소득평가액 = 실제소득 - 가구특성별 지출비용 - 근로소득공제
```

| 항목 | 포함 소득 | 공제 |
|------|----------|------|
| 근로소득 | 월급, 일용직 소득 | 30% 공제 (청년 60만원 선공제 + 30%) |
| 사업소득 | 자영업, 프리랜서 | 필요경비 공제 |
| 재산소득 | 이자, 배당 | - |
| 이전소득 | 연금, 보험금 | - |

### 재산의 소득환산액 계산

```
재산의 소득환산액 = (재산 - 기본재산공제 - 부채) × 소득환산율
```

| 재산 유형 | 소득환산율 | 예시 |
|----------|----------|------|
| 일반재산 | 월 4.17% | 부동산, 임차보증금 |
| 금융재산 | 월 6.26% | 예금, 적금, 주식 |
| 자동차 | 월 100% | 일반승용차 (예외 있음) |

| 지역 | 기본재산공제 |
|------|------------|
| 서울 | 9,900만원 |
| 경기 | 8,000만원 |
| 광역시·세종·창원 | 7,700만원 |
| 그 외 | 5,300만원 |

## 3단계: 비율 확인

### 계산 공식

```
내 중위소득 비율(%) = (소득인정액 ÷ 가구원수별 기준 중위소득) × 100
```

### 예시: 1인 가구, 소득인정액 130만원

```
130만 ÷ 256.4만 × 100 = 약 50.7%
→ 중위소득 약 51%
```

### 예시: 4인 가구, 소득인정액 300만원

```
300만 ÷ 649.5만 × 100 = 약 46.2%
→ 중위소득 약 46%
→ 주거급여(48%) 이하, 교육급여(50%) 이하 해당
```

### 비율별 받을 수 있는 복지

| 비율 | 주요 복지 |
|------|----------|
| 32% 이하 | 생계급여, 의료급여, 주거급여, 교육급여 |
| 40% 이하 | 의료급여, 주거급여, 교육급여 |
| 48% 이하 | 주거급여, 교육급여 |
| 50% 이하 | 교육급여, 차상위 감면혜택 |
| 60% 이하 | 한부모지원, 청년월세, 국민취업 1유형 |
| 80% 이하 | 긴급복지지원, 드림스타트 |

## 자주 묻는 질문

### Q. 월급만 보면 안 되나요?

안 됩니다. 소득인정액은 월급(근로소득) + 사업소득 + 재산소득 + 이전소득에서 공제를 빼고, 재산의 소득환산액까지 더한 금액이에요. 월급이 적어도 재산이 많으면 비율이 높아질 수 있습니다.

### Q. 부모와 같이 살면 부모 소득도 합산하나요?

같은 가구로 판단되면 합산합니다. 주민등록등본상 같은 주소에 있고 생계를 같이하면 한 가구예요. 다만 성인 자녀가 독립 가구임을 증빙하면 분리할 수 있습니다.

### Q. 자동차가 있으면 무조건 탈락하나요?

아닙니다. **배기량 2000cc 미만** 또는 **차량가액 500만원 미만** 자동차는 일반재산으로 환산됩니다. 장애인·생업용 차량도 예외가 있어요. 2000cc 이상 고가 차량만 100% 환산이 적용됩니다.

### Q. 전세보증금도 재산에 포함되나요?

네. 임차보증금은 일반재산에 포함됩니다. 다만 기본재산공제(서울 9,900만원 등)를 차감하므로, 보증금이 기본공제 이하면 소득환산액이 0원이에요.

### Q. 온라인으로 정확하게 확인할 수 있나요?

복지로 모의계산기가 가장 정확합니다. 실제 신청 시에는 주민센터에서 공적 자료 조회를 통해 최종 판정하므로 모의계산과 차이가 있을 수 있어요.
