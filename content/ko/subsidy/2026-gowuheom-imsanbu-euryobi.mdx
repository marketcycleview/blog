고위험 임신으로 입원하셨나요?

정부에서 고위험 임산부에게 **최대 300만원**까지 의료비를 지원합니다. 2024년부터 **소득 기준이 폐지**되어 누구나 받을 수 있어요.

## 지원 대상

### 대상 조건

**19대 고위험 임신질환**으로 진단받고 **입원치료**를 받은 임산부

| 조건 | 내용 |
|------|------|
| 소득 기준 | **없음** (2024년부터 폐지) |
| 치료 형태 | **입원치료** |
| 신청 기한 | 분만일로부터 **6개월 이내** |

> **중요**: 외래 진료비는 지원 대상이 아닙니다. **입원치료비만** 지원됩니다.

### 19대 고위험 임신질환

다음 질환으로 입원 치료한 경우 지원받을 수 있습니다.

| 번호 | 질환명 |
|------|--------|
| 1 | 조기진통 |
| 2 | 분만관련 출혈 |
| 3 | **중증 임신중독증** |
| 4 | 양막의 조기파열 |
| 5 | 태반조기박리 |
| 6 | **전치태반** |
| 7 | 절박유산 |
| 8 | 양수과다증 |
| 9 | 양수과소증 |
| 10 | 분만전 출혈 |
| 11 | **자궁경부무력증** |
| 12 | 고혈압 |
| 13 | **다태임신** (쌍둥이 등) |
| 14 | 당뇨병 |
| 15 | 대사장애를 동반한 임신과다구토 |
| 16 | 신질환 |
| 17 | 심부전 |
| 18 | **자궁내 성장제한** |
| 19 | 자궁 및 자궁 부속기 질환 |

> **참고**: 신질환, 심부전은 O코드(임신, 출산 및 산후기)가 진단서에 함께 기재되어야 합니다.

### 사산도 지원 대상

분만 결과 **사산**한 경우에도 지원 대상에 포함됩니다.

## 지원 금액

| 항목 | 내용 |
|------|------|
| 지원 비율 | 입원치료비의 **90%** |
| 지원 한도 | 최대 **300만원** |
| 소득 기준 | **없음** |

### 지원 항목

| 지원 O | 지원 X |
|--------|--------|
| 진찰료 | 병실 입원료 |
| 투약/조제료 | 환자 특식 |
| 주사료 | 상급병실 차액 |
| 처치/수술료 | - |
| 검사료 | - |

> 건강보험 급여 중 **전액본인부담금 + 비급여 진료비**의 90%를 지원합니다.

## 지원 예시

### 조기진통으로 2주 입원한 경우

| 항목 | 금액 |
|------|------|
| 총 입원비 | 500만원 |
| 건강보험 부담 | 350만원 |
| 본인부담 (비급여 포함) | 150만원 |
| **지원금 (90%)** | **135만원** |

## 신청 방법

### 신청 기한

**분만일로부터 6개월 이내**에 신청해야 합니다.

### 1. 보건소 방문 신청

**주민등록 주소지 관할 보건소**에 방문합니다.

### 2. 온라인 신청

**e보건소 공공보건포털** 또는 **아이마중 앱**에서 신청 가능합니다.

1. [e보건소](https://www.e-health.go.kr) 접속
2. 고위험임산부 의료비 지원 메뉴 선택
3. 온라인 신청서 작성 및 서류 첨부

## 자주 묻는 질문

### Q. 소득 기준이 없어졌다는데 맞나요?

네, **2024년부터 소득 기준이 폐지**됐습니다. 고소득자도 신청 가능해요.

### Q. 외래 진료비도 지원되나요?

아니요, **입원치료비만** 지원됩니다. 외래 진료비는 대상이 아니에요.

### Q. 여러 번 입원하면 각각 지원받을 수 있나요?

네, 입원 횟수별로 **각각 신청 가능**합니다. 단, 1인당 총 300만원 한도입니다.

### Q. 출산 후에 신청해도 되나요?

네, **분만일로부터 6개월 이내**에 신청하면 됩니다. 출산 후에 신청해도 돼요.

### Q. 쌍둥이 임신도 해당되나요?

네, **다태임신(쌍둥이 이상)**은 19대 고위험 질환에 포함됩니다.

<LinkButton href="https://www.e-health.go.kr/gh/caSrvcGud/selectMdclSupGudInfo.do?heBiz=PG00002&menuId=200010" label="e보건소에서 고위험임산부 의료비 신청하기" />

### Q. 국민행복카드와 중복으로 받을 수 있나요?

네, 국민행복카드(임신출산진료비)와 **중복 지원 가능**합니다.
