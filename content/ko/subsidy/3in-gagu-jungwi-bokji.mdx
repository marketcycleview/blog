자녀 1명이 있는 3인 가구라면 기준 중위소득이 1인 가구의 **2배 이상**입니다. 그만큼 받을 수 있는 복지의 폭도 넓어져요.

2026년 3인가구 기준 중위소득은 월 **535만 9,036원**. 비율별로 어떤 복지를 받을 수 있는지 정리했습니다.

### 한부모 + 자녀 2명

한부모가구는 중위소득 **63%** 이하면 한부모가족지원을 받을 수 있습니다.

| 지원 | 금액 |
|------|------|
| 아동양육비 | 월 21만원 × 2명 = **42만원** |
| 추가양육비 (5세 이하) | 월 5만원 × 해당 자녀 |
| 생계비 (청소년한부모) | 월 35만원 |
| 교육비 | 연 9.3만원 (중학생), 5.4만원 (고등학생) |

## 3인가구 소득인정액 예시

### 맞벌이 부부 + 자녀 1명, 경기도

| 항목 | 금액 |
|------|------|
| 남편 근로소득 | 300만원 |
| 아내 근로소득 | 180만원 |
| 근로소득공제 (각 30%) | -144만원 |
| 소득평가액 | 336만원 |
| 재산 (전세 1.5억) | 1.5억 |
| 기본공제 (경기) | -8,000만원 |
| 재산환산 | 291.9만원 |
| **소득인정액** | **약 628만원** |

```
중위소득 비율: 628만 ÷ 536만 × 100 = 약 117%
→ 기초급여 해당 안 됨
→ 아동수당, 보육료 등 소득 무관 혜택은 가능
```

### 한부모 + 자녀 2명, 근로소득 150만원

| 항목 | 금액 |
|------|------|
| 근로소득 | 150만원 |
| 근로소득공제 (30%) | -45만원 |
| 소득평가액 | 105만원 |
| 재산환산 | 0원 (기본공제 이내 가정) |
| **소득인정액** | **약 105만원** |

```
중위소득 비율: 105만 ÷ 536만 × 100 = 약 19.6%
→ 생계급여(32%) 이하 → 4대 급여 전부 수급
→ 한부모가족지원 추가 (63% 이하)
→ 월 혜택: 생계급여 약 161만 + 주거급여 + 한부모 42만...
```

## 자주 묻는 질문

### Q. 자녀가 태어나면 가구원수가 바뀌나요?

네. 출생 신고 후 가구원수가 변경되면 기준 중위소득도 올라갑니다. 2인→3인이 되면 기준이 약 116만원 올라가서 수급 자격이 유리해져요.

### Q. 3인가구면 아이가 있어야 하나요?

아닙니다. 부모 2명 + 성인 자녀 1명, 또는 형제·자매 3명 등도 3인 가구가 될 수 있어요. 가구원수만 중요하지 구성은 상관없습니다.

### Q. 맞벌이 3인가구는 복지 받기 어렵나요?

맞벌이면 소득이 높아지지만, 3인가구 기준도 536만원으로 높습니다. 재산이 적고 소득이 중간 수준이면 60%~80% 구간의 복지는 충분히 가능해요.

### Q. 한부모가구는 별도 기준인가요?

한부모가족지원은 중위소득 63% 기준이에요. 기초생활보장 급여와 별도로 한부모가족지원을 **추가로** 받을 수 있습니다.
