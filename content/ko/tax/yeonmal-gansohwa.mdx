홈택스 연말정산 간소화서비스, 클릭 몇 번이면 끝납니다.

의료비, 교육비, 카드 사용액 등 공제 자료를 한 번에 조회하고 PDF로 다운로드할 수 있는 서비스인데요. 올해는 헬스장·수영장 이용료 등 **3종이 추가**되어 총 45종의 자료를 제공합니다.

아래에서 PC(홈택스)와 모바일(손택스) 이용법을 단계별로 정리했습니다.

---

## 핵심 정보

| 항목 | 내용 |
|------|------|
| 서비스 오픈 | 2026년 1월 15일 |
| 확정 자료 조회 | 2026년 1월 20일 이후 권장 |
| 이용 채널 | 홈택스(PC), 손택스(모바일) |
| 제공 자료 | 총 45종 (의료비, 교육비, 카드, 보험료 등) |
| 비용 | 무료 |

> 1월 15~20일은 접속자가 몰립니다. **1월 21일(수) 이후**에 이용하면 대기 없이 빠르게 조회할 수 있어요.

---

## PC(홈택스) 이용 방법

### Step 1. 홈택스 접속 및 로그인

[홈택스](https://hometax.go.kr) 접속 → 우측 상단 **로그인** 클릭

로그인 방법은 4가지입니다:

| 인증 방식 | 특징 |
|----------|------|
| 공동인증서 | 은행/공공기관 발급 인증서 |
| 금융인증서 | 금융결제원 발급 (클라우드 저장) |
| 간편인증 | 카카오톡, 네이버, PASS, 토스 등 |
| 아이디/비밀번호 | 사전 등록 필요 |

간편인증이 가장 편합니다. 카카오톡이나 네이버로 로그인하면 별도 인증서 없이 바로 이용 가능해요.

### Step 2. 연말정산 간소화 메뉴 진입

로그인 후 상단 메뉴에서:

**장려금·연말정산·전자기부금** → **연말정산간소화** → **소득·세액공제 자료 조회**

메인 화면에 '연말정산 간소화' 바로가기 배너가 뜰 수도 있습니다. 배너를 클릭해도 같은 페이지로 이동해요.

### Step 3. 귀속 연도·근무 기간 확인

| 확인 항목 | 설명 |
|----------|------|
| 귀속 연도 | **2025년** 선택 |
| 근무 기간 | 계속 근무자는 1~12월 전체 선택 |

중도입사나 이직자는 **실제 근무한 달만** 선택해야 합니다. 근무하지 않은 기간의 자료는 공제 대상이 아니에요.

### Step 4. 항목별 자료 조회

화면에 공제 항목이 아이콘으로 나열됩니다. 각 항목의 **돋보기 아이콘**을 클릭하면 해당 내역이 조회돼요.

**주요 조회 항목**:

| 항목 | 내용 |
|------|------|
| 건강보험 | 건강보험료 납부 내역 |
| 국민연금 | 국민연금 납입액 |
| 보험료 | 보장성보험, 장애인전용보험 |
| 의료비 | 병원·약국·안경 등 |
| 교육비 | 학교, 학원, 교복 등 |
| 신용카드 | 신용카드·체크카드·현금영수증 |
| 주택자금 | 주택청약, 전세대출 이자 등 |
| 연금저축 | 연금저축·IRP 납입액 |
| 기부금 | 정치자금, 법정/지정 기부금 |
| 월세 | 월세 납입 내역 (등록된 경우) |

각 항목을 하나씩 클릭해서 **금액이 맞는지 확인**하세요. 간소화 자료는 영수증 발급기관이 제출한 자료 그대로 표시되기 때문에, 실제와 다를 수 있습니다.

### Step 5. PDF 다운로드

모든 항목 확인 후 **한번에 내려받기** 버튼을 클릭합니다.

- PDF 파일로 저장됨
- 이 파일을 회사 경리/인사팀에 제출
- 회사에서 자체 시스템(ERP 등)을 쓴다면 업로드 방식이 다를 수 있음

> PDF 파일명에 주민등록번호가 포함되어 있으니 보관에 주의하세요.

## 모바일(손택스) 이용 방법

PC 없이 스마트폰으로도 동일하게 이용할 수 있습니다.

### Step 1. 손택스 앱 설치

앱스토어(iOS) 또는 플레이스토어(Android)에서 **'국세청 손택스'** 검색 → 설치

### Step 2. 로그인

간편인증(카카오톡, 네이버, PASS 등)으로 로그인하면 가장 빠릅니다. 지문인증을 등록해두면 다음번에 더 편해요.

PC에 있는 공동인증서를 모바일로 옮기려면 손택스 로그인 화면에서 **'인증서 가져오기'**를 선택하면 됩니다.

### Step 3. 간소화 메뉴 진입

전체메뉴 → **장려금·연말정산·전자기부금** → **연말정산간소화** → **근로자 소득세액공제 자료조회**

### Step 4. 자료 조회 및 다운로드

PC와 동일하게 귀속 연도·월 선택 → 항목별 조회 → PDF 다운로드까지 가능합니다.

다운로드한 PDF는 카카오톡이나 이메일로 바로 전송할 수 있어서, 회사 담당자에게 보내기 편합니다.

## 올해 새로 추가된 자료 3종

2026년 간소화서비스에 **3종이 새로 추가**되어 총 45종이 됐습니다.

| 추가 항목 | 내용 |
|----------|------|
| 수영장·헬스장 이용료 | 2025.7.1 이후 결제분 (문화비 소득공제) |
| 발달재활서비스 이용증명 | 장애아동 재활서비스 비용 |
| 장애인활동지원급여 본인부담금 | 장애인활동지원 서비스 비용 |

헬스장·수영장 이용료는 **총급여 7,000만원 이하** 근로자가 대상이고, 30% 소득공제를 받을 수 있습니다.

## 부양가족 자료 조회 (동의 절차)

본인 외에 **부양가족 자료**를 조회하려면 사전 동의가 필요합니다.

### 동의가 필요 없는 경우

- **만 19세 미만 자녀**: 부모가 직접 조회 가능 (별도 동의 불필요)

### 동의가 필요한 경우

- 성인 자녀, 배우자, 부모님 등 → **본인이 직접 동의**해야 함

**온라인 동의 방법**:
1. 부양가족 본인이 홈택스 또는 손택스 로그인
2. '연말정산 간소화' → '자료제공 동의' 메뉴
3. 제공 대상자(근로자) 정보 입력 → 동의 완료

**오프라인 동의 방법 (고령 부모님 등)**:

| 순서 | 내용 |
|------|------|
| 1 | 홈택스에서 '소득·세액 공제 정보제공 동의 신청서' 다운로드 |
| 2 | 신청서 작성 + 부양가족 신분증 사본 + 가족관계증명서 준비 |
| 3 | 국세청 팩스(1544-7020)로 전송 또는 세무서 방문 제출 |

처리까지 1~3일 정도 걸립니다. 미리 해두는 게 좋아요.

## 간소화에서 빠지는 자료 (직접 준비)

간소화서비스가 편리하긴 하지만, **모든 자료가 자동으로 뜨는 건 아닙니다**.

| 누락 가능 항목 | 해결 방법 |
|---------------|----------|
| 안경·콘택트렌즈 | 안경점에서 영수증 직접 발급 |
| 보청기·장애인 보장구 | 구입처에서 영수증 발급 |
| 월세 납입 내역 | 계좌이체 내역 + 임대차계약서 준비 |
| 일부 기부금 | 기부 단체에서 영수증 발급 |
| 해외 교육비 | 해당 교육기관 영수증 |
| 종교단체 기부금 | 종교단체 기부금영수증 |

> 간소화 자료에 뜨더라도 **금액이 정확한지 반드시 확인**하세요. 영수증 발급기관이 잘못 제출한 자료가 그대로 표시될 수 있습니다.

## 일괄제공 서비스 (회사에서 자동 수집)

회사에서 간소화 자료를 일괄로 내려받을 수 있는 서비스입니다. 근로자가 직접 PDF를 다운받아 제출하는 수고를 줄여줘요.

### 이용 조건

| 항목 | 내용 |
|------|------|
| 회사 | 국세청에 근로자 명단 등록 (11월 말까지) |
| 근로자 | 홈택스에서 일괄제공 동의 (1월 19일까지) |
| 자료 제공 | 1월 20일부터 회사에서 일괄 다운로드 |

같은 회사에 계속 근무 중이면 **최초 1회만 동의**하면 됩니다. 신입·이직자만 새로 동의하면 돼요.

동의 기한(1월 19일)을 놓쳤다면 본인이 직접 PDF를 다운받아 제출하면 됩니다.

## 자주 묻는 질문

### Q. 간소화서비스에 나오는 금액이 실제와 다릅니다

영수증 발급기관이 국세청에 제출한 자료가 그대로 표시되는 거라, 오류가 있을 수 있어요. 이 경우 해당 기관에 직접 확인하고, 정확한 영수증을 별도로 제출하면 됩니다.

### Q. 조회가 안 되는 항목이 있어요

1월 20일 이전이면 아직 자료가 업데이트되지 않았을 수 있습니다. 20일 이후에 다시 확인해보세요. 그래도 안 뜨는 항목은 영수증 발급기관에서 직접 발급받아야 합니다.

### Q. 부양가족 동의를 안 했는데 조회할 수 있나요?

만 19세 미만 자녀 외에는 불가능합니다. 부양가족 본인이 직접 홈택스에서 동의하거나, 오프라인 신청(팩스/방문)을 해야 해요.

### Q. 이직했는데 어떻게 하나요?

현재 재직 중인 회사에서 연말정산을 합니다. 전 직장의 원천징수영수증을 받아서 현 직장에 제출하면 돼요. 간소화서비스 자료는 전 직장 근무 기간을 제외하고 조회하세요.

### Q. 손택스와 홈택스 자료가 다른가요?

동일합니다. 같은 자료를 PC와 모바일에서 조회하는 것뿐이에요.
