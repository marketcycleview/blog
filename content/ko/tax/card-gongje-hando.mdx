카드로 3,000만원을 써도 공제는 최대 600만원입니다. 한도가 있거든요.

"열심히 썼는데 왜 더 안 올라가지?" 싶을 때, 대부분 기본 한도에 걸린 겁니다. 기본 한도 외에 **추가 한도**가 따로 있다는 걸 모르면 놓치는 공제도 생겨요. 한도 구조를 알면 같은 소비로도 공제를 더 받을 수 있습니다.

---

## 카드 공제 한도 구조

카드 소득공제 한도는 **기본 한도 + 추가 한도** 두 층으로 되어 있습니다.

| 구분 | 한도 | 대상 |
|------|------|------|
| 기본 한도 | 200~300만원 | 신용카드·체크카드·현금영수증 합산 |
| 추가 한도 | 각 100만원 | 전통시장·대중교통·문화비 별도 |

기본 한도를 다 채워도 추가 한도는 별도로 적용되니까, **최대 600만원**까지 가능합니다.

---

## 기본 한도: 총급여에 따라 다름

| 총급여 | 기본 한도 |
|--------|----------|
| **7,000만원 이하** | **300만원** |
| 7,000만원 초과 ~ 1.2억원 | **250만원** |
| 1.2억원 초과 | **200만원** |

연봉이 높을수록 한도가 줄어드는 구조입니다. 총급여 7,000만원 이하가 가장 유리해요.

### 총급여별 기본 한도 도달에 필요한 카드 사용액

기본 한도를 꽉 채우려면 문턱(25%) + 한도분만큼 써야 합니다.

| 총급여 | 25% 문턱 | 기본 한도 300만원 채우려면 (신용카드 기준 15%) |
|--------|----------|----------------------------------------------|
| 3,000만원 | 750만원 | 750 + 2,000 = **2,750만원** |
| 4,000만원 | 1,000만원 | 1,000 + 2,000 = **3,000만원** |
| 5,000만원 | 1,250만원 | 1,250 + 2,000 = **3,250만원** |
| 6,000만원 | 1,500만원 | 1,500 + 2,000 = **3,500만원** |

> 신용카드만으로 기본 한도를 채우려면 상당한 금액이 필요합니다. 체크카드(30%)를 섞으면 더 적은 금액으로 채울 수 있어요.

---

## 추가 한도: 기본 한도와 별도

기본 한도를 다 채웠어도 아래 항목은 **추가로** 공제받을 수 있습니다.

| 추가 한도 항목 | 공제율 | 추가 한도 |
|--------------|--------|----------|
| **전통시장** | 40% | **+100만원** |
| **대중교통** | 40% | **+100만원** |
| **도서·공연·영화** | 30% | **+100만원** (총급여 7,000만원 이하만) |

### 추가 한도 계산 예시

총급여 5,000만원, 기본 한도 300만원을 이미 채운 상태에서:

| 추가 항목 | 사용액 | 공제율 | 공제 금액 | 추가 한도 |
|----------|--------|--------|----------|----------|
| 전통시장 | 200만원 | 40% | 80만원 | 100만원 이내 ✅ |
| 대중교통 | 150만원 | 40% | 60만원 | 100만원 이내 ✅ |
| 도서·영화 | 100만원 | 30% | 30만원 | 100만원 이내 ✅ |
| **추가 공제 합계** | | | **170만원** | |

기본 300만원 + 추가 170만원 = **총 470만원 공제**.

## 최대 공제 시나리오

총급여 5,000만원 직장인이 받을 수 있는 최대 카드 공제:

| 한도 구분 | 최대 금액 |
|----------|----------|
| 기본 한도 | 300만원 |
| 전통시장 추가 | 100만원 |
| 대중교통 추가 | 100만원 |
| 문화비 추가 | 100만원 |
| **합계** | **600만원** |

600만원 공제를 받으면 세율 15% 구간에서 **약 90만원**, 24% 구간에서 **약 144만원** 절세됩니다.

## 추가 한도 항목 상세

### 전통시장 (공제율 40%, 추가 한도 100만원)

| 인정되는 곳 | 인정 안 되는 곳 |
|-----------|---------------|
| 전통시장 내 점포 | 대형마트 |
| 재래시장 | 백화점 |
| 시장 내 음식점 | 편의점 |

전통시장 여부는 **소상공인시장진흥공단**에 등록된 시장 기준입니다. 카드 결제 시 자동으로 전통시장으로 분류돼요.

### 대중교통 (공제율 40%, 추가 한도 100만원)

| 인정되는 교통수단 | 인정 안 되는 교통수단 |
|----------------|-------------------|
| 지하철·전철 | 택시 |
| 시내·시외버스 | KTX·SRT (고속철도는 인정) |
| 고속버스 | 비행기 (항공권) |
| KTX·SRT | 자가용 유류비 |

> KTX·SRT도 대중교통으로 인정됩니다. 출장이나 귀성 시 이용한 고속철도 비용도 40% 공제율이 적용돼요.

### 도서·공연·영화 (공제율 30%, 추가 한도 100만원)

| 항목 | 인정 여부 |
|------|----------|
| 서점에서 책 구입 | ✅ |
| 온라인 서점 (교보, 예스24 등) | ✅ |
| 전자책 (e-book) | ✅ |
| 공연 티켓 (뮤지컬, 연극, 콘서트) | ✅ |
| 영화 티켓 | ✅ |
| OTT 구독료 (넷플릭스 등) | ❌ |
| 학습지·참고서 | ❌ (교육비 성격) |

**총급여 7,000만원 이하**인 근로자만 해당됩니다. 7,000만원 초과자는 문화비 추가 한도를 받을 수 없어요.

---

## 한도 초과 시 어떻게?

기본 한도를 다 채웠는데 추가 한도 항목도 없다면, 더 이상 카드 공제를 늘릴 수 없습니다.

### 한도 초과 확인 방법

홈택스 **'연말정산 미리보기'** 서비스(10월 말~)에서 현재까지 카드 사용 현황과 예상 공제액을 확인할 수 있어요.

```
홈택스 → 편리한 연말정산 → 연말정산 미리보기
→ 신용카드 등 소득공제 예상액 확인
```

### 한도 초과 시 전략

| 전략 | 설명 |
|------|------|
| **전통시장으로 소비 이동** | 기본 한도 외 추가 100만원 |
| **대중교통 적극 이용** | 기본 한도 외 추가 100만원 |
| **도서·공연·영화 소비** | 기본 한도 외 추가 100만원 |
| **다른 공제 항목 확인** | 연금저축, IRP, 기부금 등 |

기본 한도를 채웠다면 추가 한도 항목으로 이동하고, 추가 한도까지 채웠다면 카드가 아닌 다른 공제를 늘리는 게 효율적입니다.

## 맞벌이 부부 한도 전략

맞벌이 부부는 각자 기본 한도를 갖고 있습니다.

| | 남편 (연봉 6,000만원) | 아내 (연봉 4,000만원) |
|------|-----|-----|
| 기본 한도 | 300만원 | 300만원 |
| 추가 한도 | 각 100만원 × 3 | 각 100만원 × 3 |
| 최대 합계 | 600만원 | 600만원 |

부부 합산 최대 **1,200만원**까지 공제 가능합니다. 다만 각자 본인 명의 카드만 공제되므로, 소비를 한쪽에 몰아도 한 사람의 한도를 넘기면 의미가 없어요.

### 팁: 한도 미달인 쪽의 소비를 늘리기

남편이 기본 한도를 다 채웠는데 아내 쪽은 문턱도 안 넘겼다면, 아내 명의 카드로 소비를 옮기는 게 전체적으로 유리합니다.

## 총급여별 최적 전략 정리

| 총급여 | 기본 한도 | 핵심 전략 |
|--------|----------|----------|
| 3,000만원 이하 | 300만원 | 문턱 자체가 낮아 한도 채우기 쉬움. 체크카드 비중 늘리기 |
| 4,000~5,000만원 | 300만원 | 문턱까지 신용카드 → 이후 체크카드 전환 |
| 6,000~7,000만원 | 300만원 | 문턱이 높아 한도 채우기 어려울 수 있음. 전통시장·대중교통 활용 |
| 7,000만원 초과 | 250만원 | 한도가 줄어듦. 추가 한도 항목 적극 활용 |
| 1.2억 초과 | 200만원 | 한도 200만원으로 축소. 다른 공제 항목에 집중 |

## 자주 묻는 질문

### Q. 기본 한도 300만원을 못 채웠는데 추가 한도도 받을 수 있나요?

네. 추가 한도는 기본 한도와 별개입니다. 기본 한도를 채우지 않았어도 전통시장·대중교통 사용분은 추가 한도 범위에서 공제받을 수 있어요. 다만 실질적으로 문턱(25%)을 넘은 상태여야 합니다.

### Q. 전통시장에서 신용카드를 쓰면 40%인가요 15%인가요?

전통시장 사용분은 결제 수단과 관계없이 **40%** 공제율이 적용됩니다. 신용카드로 결제해도 40%예요.

### Q. 대중교통 정기권도 공제 대상인가요?

네. 대중교통 정기권(지하철, 버스)도 대중교통 공제 40%가 적용됩니다. 교통카드나 카드 결제 어느 쪽이든 간소화에 반영돼요.

### Q. 문화비 공제에 OTT 구독료는 왜 안 되나요?

도서·공연·영화 공제는 **문화체육관광부에 등록된** 도서·공연·영화관에서의 소비만 해당됩니다. OTT(넷플릭스, 왓챠 등)는 현재 이 범위에 포함되지 않아요.

### Q. 택시비도 대중교통 공제 되나요?

안 됩니다. 택시는 대중교통 공제 대상에 포함되지 않아요. 버스, 지하철, 고속버스, KTX 등만 해당됩니다.
